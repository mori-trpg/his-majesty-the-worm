> 「虎，虎，烈火般明燃，
> 在黑夜的林間，
> 是何不朽之手與眼，
> 造就了你懾人的對稱？」
>
> ──威廉·布萊克，〈老虎〉

廣闊世界幅員遼闊，棲居著各式奇異生物、異國文化、宗教分裂、相互征戰的氏族，以及源遠流長的古老血脈。即便人類有著無盡的變化，精靈中也沒有兩個完全相同的個體。從萊馬克的獨居巨魔，到阿爾夫海姆的高貴精靈，到蹂躪鬼草平原的蠻族部落，到彌思里斯的堅忍聖堂騎士，廣闊世界中有許多民族——他們都聚集在城市中，渴望在地下世界追求榮耀與財富。

學識不存在。
閱讀本章時，你會注意到這裡有許多奇幻架空的說法。若這些內容能給你靈感，就以這些概念為基礎，為你的角色發揮。若它讓你感到困惑、受限，或與你自己的想法相矛盾，就忽略它。用麥克筆把它劃掉。在書的空白處寫下你的想法。讓你的遊戲世界成為你自己的。
