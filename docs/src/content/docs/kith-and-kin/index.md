廣闊世界幅員遼闊，棲居著各式奇異生物、異國文化、宗教分裂、相互征戰的氏族，以及源遠流長的古老血脈。即便人類有著無盡的變化，精靈族中也沒有兩個完全相同的個體。從萊馬克的獨居巨魔，到阿爾夫海姆的高貴精靈，到蹂躪鬼草平原的蠻族部落，到彌思里斯的堅忍聖堂騎士，廣闊世界中有許多民族——他們都聚集在城市中，渴望在地下世界追求榮耀與財富。

人類

驕傲何時變成妄自尊大？榮耀何時蛻變為虛榮？這條細線體現在人類身上——廣闊世界中的無賴與君王。

人類不如永生的精靈族強韌，容易罹患疾病與衰老。他們也不如地底民狡黠，在煉金術與自然哲學知識方面發展遲緩。然而，人類有一項超越所有其他種族的天賦：戰爭。

人類天生為戰爭而生。他們從矮人與精靈那裡習得的工藝，都被巧妙地轉化為這個目的。他們繁殖快速、成長迅速，不斷為城市補充男男女女，替換那些在苦難中逝去的人。他們兇猛、頑強、勇敢。憑藉戰爭技藝，人類已征服了廣闊世界的大部分地區。他們是征服的君王，驕傲而可畏。

即便如此，他們也紛爭不斷。人類從未在同一面旗幟下團結（儘管各路軍閥都曾嘗試這一壯舉）。哲學、宗教或血緣的分歧都會令他們分裂——野心亦然。因此，人類的力量分散於數個顯赫帝國和數百個小王國之中。

儘管如此，人類展現了許多值得稱許的特質。他們是忠誠的民族，視忠信為最崇高的美德。他們在歌謠與故事中長久銘記父輩的事蹟，並努力效法。這反過來又激發了他們為家族贏得榮耀與聲名的動力。這種對聲望的渴望強化了他們的社群，因為他們因害怕給血脈帶來恥辱而摒棄不肖之輩。確實，沒有幾個民族比人類更強烈地保護自己的族裔與親屬。他們有著極大的善意，儘管也有著同樣的兇悍。

人類的神話
一元神是一。一元神是多。

如同繁星散布的天空有著許多星座；如同建築物有著許多側面；一元神有著許多面貌，皆映照出其統一性的更大真理。

是先知彌思里斯教導了我們這些奧義（讚頌歸於他）。是智慧聖女賦予了他生命（祝福歸於她的名字）。

彌思里斯教導我們，被不同大家族所崇拜的諸神，都是唯一真神的流溢。這唯一的真神被稱為一元神。

彌思里斯教導說，一元神有二十一個獨立的流溢。因此，雖然少女可能向戀人牌祈求愛情的庇佑，武士可能向力量牌祈求戰鬥的幫助，但所有的祈禱與榮耀最終都歸向祂。因為人類的感知有限，在只有一個面貌的地方，他們看到了許多。

一元神是魔術師，遠古時代曾向人類揭示魔法的運作。
祂也是女祭司，治癒病患，護佑兒童與母親。
祂也是御車者，馳騁夜空，引導水手前行。
祂是月亮之神。
祂是太陽女神。
祂是死神。
祂是世界。
祂是惡魔，那個造出一切，卻造得一塌糊塗的存在。

人類血脈
人類將世界分為兩部分：我族與他族。他們透過建立偉大而高貴的家族來做到這一點。他們以婚姻、愛情與誓言緊密相連。他們在要塞上飄揚著美麗的旗幟，在工藝品上展示著驕傲的紋章盾徽。他們從小就被灌輸對特定美德的尊重。即便是出身最低微、境況最悲慘的奴隸孩子，也生於某面旗幟飄揚之處。同屬一個家族的兩個人類同屬一脈。他們可能並無直接血緣關係，但很可能彼此相識。
