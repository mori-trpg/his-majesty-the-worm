## 獸人

「致Rng的祈禱：『喔，Rng，慈悲而仁善。我們已擁有所需的一切。您的子民滿足而快樂。我們的生活充實，別無所求。懇請您將恩賜降於我們的敵人。給予他們所缺少的。讓他們感受您的愛與慷慨。』」

在自己的土地上，獸人與其他人並無二致。他們有家庭，有農場，喜歡與朋友舉起蜂蜜酒角豪飲。他們在宗教節慶上製作手工藝品販售。他們為生病的山羊憂心忡忡。然而，幾乎沒有人看見獸人的這一面。大多數人只看見入侵者。

據說獸人腸中燃著一把火，唯有黃金的存在才能澆熄。沒有黃金，獸人確實如瘋如狂。為何他們要擄走王族？沒有人知道。有時他們向貴族家族索取大筆贖金。但有時候也不這麼做。

獸人是著名的劫掠者和商人。他們遊歷廣闊世界各地，是最出色的航海者。在夏季，當他們的土地與人類王國之間的冰封海峽融化，成群的獸人劫掠者便會來犯。他們掠奪黃金與王族。

奴隸制度違反獸人法典（一套口傳的法律與習俗集成），因此永久留在獸人土地上的王族通常會被融入社會。有時他們在冰封的北方度過餘生；有時他們被十字軍救回。大約一百年前，各貴族家族曾發動三次主要遠征（以及數次小型遠征），試圖從獸人手中奪回這些被擄走的人。這些遠征在當今人類與獸人的心中都記憶猶新。

獸人大多奉行母系制度，這是因為部分獸人沒有父親。獸人可以透過孤雌生殖進行無性繁殖。這種繁殖方式（奇特的是）既能產生雄性獸人，也能產生雌性獸人；每個獸人都是母親的半克隆體。

獸人的幼崽被稱為哥布林。不同於人類孩子需要數年才能學會走路和說話，哥布林學習這些技能的速度驚人。在短短數天內，一個哥布林嬰兒便能站立行走、開口說話、打架搗亂、撬鎖開門。說真的，這些小傢伙實在了不起。

並非出於殘忍，但獸人對自己後代健康的關注相當淡然。很少有哥布林能長大成為獸人。這是個艱難的世界。對獸人父母來說，能活得夠久以再生育一窩哥布林，比起讓單個哥布林（甚至整窩哥布林）存活更為重要。優先保護長者。

### 龍的秘密
獸人生命週期的最後一個階段，是蛻變為龍。就像很少有哥布林能存活成為獸人，也很少有獸人能活得夠久以蛻變為龍。這種改變只有在獸人年約百歲時才會出現，且需要漫長的時間。獸人開始「築巢」——向地下挖掘，設置幾個陷阱，然後開始長達數十年的緩慢蛻變，逐漸化為一條龍。當冒險者發現「龍蛋」時，其實只不過是在尋找「獸人的繭」。

就連獸人自己也不知道自己會蛻變成龍。他們的社會崇敬龍，但這種蛻變要麼太過禁忌，要麼太過罕見，從未在獸人社會中被討論。

「我們崇拜Rng，並非出於虔誠，而是出於恐懼。

牠的聲音是降雨的聲音，那雨既能滋潤你的莊稼，也能淹沒你的田地。

牠的氣息是風，那風既攜帶玫瑰的芬芳，也助長森林之火。

牠曾聽聞一個飢餓民族的哭號，為他們差遣了一頭能永恆供應肉食的野獸。那塔洛斯克旋即摧毀了那座城鎮。

另一次，牠聽見了在林中迷路之人的祈禱，將他們安然引導至最近的營地。那些迷路者不久後便死於敵軍刀下。

牠是Rng，助人者，亦是害人者。牠是Rng，混沌之主。」

## 獸人族裔

據說，在遠古時代，獸人曾以奴隸身份侍奉龍族。獸人共有四種血裔，各自曾經侍奉四大古龍之一。後來，神祇Rng降臨，將獸人詛咒為可怕的自由。

注意：所有獸人族裔天賦均以承受傷勢為觸發條件。[冒險者進階](/adventurer/advancement/)的傷勢說明。

### 大地血裔
大地血裔獸人擁有綠色的皮膚。他們在地下時感到最為平靜，偏好棲居洞穴，而非建造傳統的草皮屋頂房屋。

地底民最常與大地血裔相遇。其他血裔通常向南航行前往人類的土地，而大地血裔獸人則在地底民的居住區域挖掘隧道，取得了極大的成功。他們偏好盜墓（惱人的矮人較少主動揮斧砍你），但若有機會順手從半身人那裡拿走黃金，他們也毫不介意。

### 大地血裔獸人天賦：水銀之血
你的血液外觀與質地宛如水銀，對金屬具有極強的腐蝕性。當此天賦承受傷勢時，若傷害來源是金屬武器，該武器將被摧毀。

### 海洋血裔
黑色的皮膚宛如深海或滿載雨水的雲朵，海洋血裔獸人能在心中深切感受到潮汐的漲落。海洋血裔獸人在海岸邊或廣闊水域上棲居時感到最為平靜。

雖然所有獸人都在春夏兩季劫掠人類的土地，但此血裔在海上尤為令人聞風喪膽。他們是著名的海盜，鮮少真正登陸進行劫掠或交戰。據說，不少海底人族已與海洋血裔獸人締結婚姻。

### 海洋血裔獸人天賦：毒血
你的黑色血液是致命的毒藥。當此天賦承受傷勢時，若傷害來源是撕咬，攻擊者將受到致命傷害（[傷勢與例示](/challenge-phase/wounds-and-example/)）。

### 風暴血裔
白色的皮膚宛如陰雲密布的天空，這些獸人熱情洶湧如暴風。當巨大的風暴從海上席捲而來時，他們能聽見自己內心的高歌。在狂風暴雨怒吼的時刻，他們感到最為平靜。

此血裔的先祖故土是所有獸人領土中最偏北的地方。風暴血裔忍受著最為嚴酷的冬季和激烈的暴風雪。他們聲稱，這些磨難使他們成為了獸人族群中最堅韌的一脈。確實，風暴血裔若有什麼特質，那便是長於忍耐、足智多謀、百折不撓。

### 風暴血裔獸人天賦：迷霧
你的血液如濃霧般冒煙。當此天賦承受傷勢時，你進入遮蔽狀態。此狀態持續的時間足以讓你執行一個行動，或若你保持靜止、不採取任何行動，則可無限期維持。

### 烈火血裔
烈火血裔獸人擁有深紅色與黑色的皮膚，觸摸時感覺溫熱。他們從不平靜。

烈火血裔以衝鋒陷陣的突擊效果著稱。其他血裔的獸人部落有時會帶上一隊烈火血裔充作先鋒。因此，你可能會在各種族群的獸人中發現附屬的烈火血裔傭兵。當烈火血裔進入狂暴狀態，他們能夠在訓練有素的老兵盾牆中撕開缺口。

### 烈火血裔獸人天賦：狂戰士之路
你的血液滾燙沸騰。當此天賦承受傷勢時，你可以選擇進入狂戰士之怒。在挑戰階段的狂暴狀態中，你無法執行閃避、閃躲、吟誦咒文、言語挑釁或協助他人；然而，你所有的攻擊均帶有優勢。

當你擊敗戰場上所有敵人後，狂暴狀態結束。若有盟友在此挑戰中攻擊或傷害過你，他們視同敵人計算。你必須花費決心才能提前結束狂暴。

## 至善
獸人透過執行以下事蹟獲得至善：

- 收藏一顆來自你們公會從未殺死過的怪物類別的頭骨

- 囤積一筆極具價值的寶藏

- 殺死一頭至少是你體型兩倍的怪物

當獸人完成全部三個至善觸發條件後，無論血裔為何，均獲得以下天賦：

### 獸人至善天賦：首領
你可以孵化或召集一群哥布林為你效力。哥布林是相當獨立的小傢伙。他們視你為「老大」，接受你的命令，但會用他們自己奇怪而單純的方式執行。

召集一群哥布林大軍是一個城市行動。花費任意數量的XP，吸引 2 + X 個哥布林，其中X等於你花費的XP數量。你最多只能擁有8個哥布林。

一群哥布林大軍視同一個動物同伴。無論大軍中有多少哥布林，他們始終被視為「一個生物」，一同行動。哥布林能理解你所說的任何語言，但在挑戰階段只能執行單字命令（使用命令行動下達）。他們的腦袋很小。

每當哥布林大軍承受一次傷勢，其中一名成員便會陣亡。

哥布林大軍自行攜帶食物或捕獵為食，並自備裝備。他們的裝備一塌糊塗；他們沒有任何你可以使用的物品或工具。他們可以擔任搬運者：每有一名哥布林，便能攜帶一個格位的物品。

## 獸人角色扮演概念
- 獸人基本上是重金屬迷。凡是屬於重金屬次文化的事物，獸人都熱愛。低沉的嘶吼、推擠舞池、大顆帶刺的手環、甩著長髮猛點頭——獸人樣樣具備。

- 獸人是兇猛的劫掠者與野蠻的海盜，在戰場上毫無憐憫。

- 獸人也重視家庭、講求榮譽、富有詩意、忠誠可靠，且普遍受過良好教育。

- 獸人對命運深有思考。他們認為，你的運氣是由你所做的善惡行為所造就的。一旦你做出選擇，沒有任何事物能阻止命運追上你。

- 獸人奉行混沌之神Rng的禮儀，以避免降臨倒楣的幫助。他們全天進行十數個或更多的小禁忌或儀式，試圖躲避祂的憤怒，例如：將珠子投入流動的水中、在說出願望時做出手勢避開祂的目光，以及儀式性地將用餐的一部分奉獻給祂。

- 獸人認為劫掠是光明正大的，但偷竊是不恥的。偷偷溜進某人的屋裡拿走他們的黃金是懦弱的行為。而踢開前門、刀劍相加、取走黃金、再把房子付之一炬，才是好漢作為。

- 獸人維護著漫長的口傳歷史。他們尊重詩人和行吟詩人，視他們為法律與習俗的守護者。殺死這些人是不祥之兆。

- 獸人認為魔法是女性的領域。女巫是不吉利的殺害對象。男人施法被認為是不祥的。
