《蠕蟲之王》是一款以地城探索為核心的桌上角色扮演遊戲。遊戲主持人（GM）運用本書中的工具打造一座名為地下世界的巨型地城。玩家創建冒險者，進入地下世界探索，追尋各自的個人任務。塔羅牌作為遊戲的隨機元素。

這是一款新派遊戲，卻擁有舊派精神。它的設計宗旨是讓「無聊的部分」變得有趣。食物、飢餓、光線、物品管理等生活化子系統是遊戲的核心。

遊戲擁有完善的程序：冒險者在地下世界中探險，在紮營場景中休息，再在廣闊世界中心的城市規劃長遠行動。

同伴之間的情誼稱為「羈絆」，它們驅動遊戲的休息與復原機制。角色扮演推動遊戲前進。

戰鬥遭遇由一套充滿活力的子系統處理，確保每一分鐘的戰鬥中，所有玩家都有有趣的選擇——絕無等待時間！

希望你玩得開心！

## 簡介

地下世界褻瀆著世界中心城市之下的土地。大多數遊戲以冒險者從城市踏出、跨越通往地下世界的大門的第一步開始。

## 書名的由來

本書書名取自義大利作家伊塔羅·卡爾維諾的《交叉命運城堡》（義大利語：Il castello dei destini incrociati）。這部實驗性小說描述一群旅人發現自己無法開口說話，於是藉助一副塔羅牌展示各自的過去、彼此訴說故事。由於本遊戲同樣使用塔羅牌講述故事，這個典故便顯得格外貼切。

## 為何用塔羅牌？

塔羅牌蘊含豐富的資訊。每張牌具有數字、花色、方向（正位與逆位）以及象徵意義。這些資訊是持久的。你可以將牌握在手中、從牌堆頂端抽取、從棄牌堆中取回，或面朝下放在你面前等等。這為遊戲創造了豐富的空間。

而且，塔羅牌比撲克牌看起來更酷、更有趣。

塔羅牌可在網路上或新紀元商店購得——那些販售水晶與各種奇物的神秘店鋪。你需要一副類似「經典」萊德偉特塔羅牌的牌組：共七十八張，小阿爾卡那由四組各十張的數字牌加上四張宮廷牌組成，大阿爾卡那則從 0 號排至 21 號。

## 遊戲設計理念

《蠕蟲之王》的目標是讓地城探索變得有趣。它將原本可能只是次要規則的食物、光線與時間提到核心地位，並將地城爬行的原始體驗置於聚光燈下。

### （半）人類的體驗

本遊戲聚焦於冒險者在神話般的地下世界中探索的個人體驗。

玩家扮演奇幻角色。然而，遊戲透過對基本「人類」需求的持續關注，讓這些冒險者感覺真實可信。冒險者會感到寒冷、醉酒、飢餓、慾火中燒、壓力沉重、歡欣雀躍、受傷受苦等等。這讓玩家得以沉浸於冒險者的體驗之中。

遊戲的核心後設貨幣，是冒險者彼此之間建立的羈絆。玩家與公會夥伴進行角色扮演時，便能為羈絆充能。玩家再消耗這些充能，從地城探索所積累的身心創傷中復原。

### 裁決，而非規則

歸根究底，沒有任何規則系統能完全涵蓋玩家行動的每一種可能——而這正是桌上角色扮演遊戲的樂趣所在！遊戲進行中，邊緣情況、奇特狀況與未曾設想的可能性不可避免地會出現。擁有一位 GM 來裁決這些情況，正是桌上遊戲相較於其他遊戲形式的優勢之一。因此，本書的規則作為資源呈現，並預期玩家會大量進行自製修改與改造。

當意外問題出現時，GM 可以調整規則以適應當下情況。當這種情況頻繁出現時，GM 和桌上的玩家可以共同將這一裁決轉化為更明確的固定規則。當固定規則似乎無法通過常識、真實感與趣味性的考驗時，GM 有責任做出裁決，讓規則更臻完善。

### 規模遞增的地下世界

地下世界並非一個以平衡遭遇為基礎的場所。玩家所在之處永遠不存在「適合當前等級」的預設。一條沉睡的龍對初出茅廬的隊伍來說可能是無法戰勝的守衛，只能悄悄繞過。稍有經驗的公會或許會嘗試擊敗龍以奪取她的寶藏。

無論如何，無論冒險者是新手還是老手，龍都能為他們帶來某種挑戰。

玩家應感到有能力進行實驗和冒險，並承擔風險所帶來的所有獎勵與後果。

### 致命的地下世界

遊戲——乃至冒險者卡——其實相當簡單。數字不多，特殊能力也不多。沒有「察覺檢定」——只有玩家描述他們在何處、如何搜尋。沒有「解除陷阱」技能——只有玩家描述他們如何把刀刃移過絆線，或用十呎竿在前方探路。

歸根究底，這款遊戲的核心是探索地下世界。因此，沒有任何能力可以消除地城固有的危險。

- 黑暗永遠是危險的。沒有任何冒險者能在黑暗中看得一清二楚。

- 口糧與火把永遠是珍貴的。沒有「召喚食物」的法術，而施術者召喚光明的技藝充滿危險。缺少光明或食物意味著死亡。

- 陷阱必須時刻應對。沒有「感知」或「解除」技能。每個陷阱都是需要解開的獨特謎題。

- 戰鬥永遠是致命的。治療十分費力，需要冒險者進行紮營階段。拔劍者須自行承擔風險。

### 奇想

《蠕蟲之王》中有相當多的奇想成分。這是一款清楚知道自己是角色扮演遊戲的遊戲。你扮演知道自己是冒險者的冒險者。它以輕鬆戲謔的方式對待自身的類型套路。它毫不做作。遊戲進行中，玩家與 GM 都將拆解並重建地城探索的常見套路。這應該是快樂的。這應該是令人愉悅的。
