> 「論及那拜占庭帝國，歷史幾乎一致裁定：它是迄今文明所採行的諸多形式中，最為卑劣低下、令人不齒的一種……他們擁有知識，眼前不斷展示著古希臘高貴的文學，其中充滿最崇高的英雄氣概；然而那份文學，後來在復興歐洲方面貢獻良多，卻無法在墮落的希臘人心中點燃哪怕一點高貴的火花或影子。這個帝國的歷史，不過是教士、宦官與婦人的陰謀，毒殺、密謀、一貫的忘恩負義，以及無休止的骨肉相殘所交織而成的單調故事。」
>
> ——《歐洲道德史》，威廉・愛德華・哈特波・萊基

城市是廣闊世界的中心，是創世的臍帶。它屹立於十字路口，雙面相望，凝視東西。它與阿爾夫海姆、龍之草原、八城礦穴等距。其雄偉無可匹敵，財富無與倫比。

城市曾被許多族裔佔據，但今日它是人類的根基。少數強大的人類家族持續爭奪權力，遙遠勢力與族裔的飛地在城市中搶佔立足點，在各自的使館內追求奇異的計謀。

城市裡，任何事物都可以用合適的價格購得。廣闊世界最精湛的工匠在城市中磨練技藝。對於極富有的人，甚至地下世界的力量也可以被交易。這些富有的贊助者誘使冒險者公會為了埋藏於這座大都市之下的寶藏而冒著生死之險。

如同皮膚，城市由層層疊疊的地層構成。城市的每個時代都深埋於其他時代之下。挖掘你的地窖，便能找到那些前人留下的陶片、盧恩石、神像、生鏽的機械、日記、油燈與骸骨。如同皮膚，城市也有傷疤——可怕的深坑鑽入地下世界，是美麗臉龐上的惡疾瘡口。

## 城市階段流程

當公會返回文明世界，滿載珍寶卻補給匱乏，城市階段開始。城市階段是追求長遠目標、探查下一步任務方向，並在重返冥河深淵之前補充物資的機會。本章詳述玩家在城市階段所需了解的事項。GM 主持城市階段的相關建議詳見第 190 頁，充實城市的程序詳見附錄 D：城市建構。

以下是城市階段的概要：

**1. 死亡與稅金**
每位玩家將攜帶的金幣的 50% 上繳給黑甲全衛軍，作為公會稅。

**2. 卓越事蹟**
抹去名冊上最舊的事蹟。作為整個公會，將你們上次探索中最值得記錄的成就寫入公會名冊。

**3. 城市事件**
GM 在其城市事件表上抽取一個隨機事件。

**4. 繳交委託**
公會繳交其委託任務（若有的話）。成功完成的委託任務提供金錢獎勵與 1 點經驗值。

**5. 生活費**
每位玩家支付一個生活費等級。

- 若你不支付生活費，本次城市階段你將淪為赤貧，且無法補充背包。
- 貧困生活費花費 25 金。可從臍帶市集補充貧困等級的裝備（第 162 頁）。
- 普通生活費花費 50 金。可從臍帶市集補充普通或貧困等級的裝備。可消耗充能羈絆治癒傷勢並清除狀態，如同紮營階段步驟 4（復原）（第 141 頁）。
- 奢華生活費花費 100 金。可從臍帶市集補充任意等級的裝備。治癒所有傷勢，並恢復所有決心。

**6. 城市行動**
每位玩家執行一個城市行動。

**7. 規劃下一次探索**
公會重新集合，作為整體討論後續步驟。GM 提供可能的委託清單，簽下一個（或兩個）委託。

**8. 補充地下世界**
在下一次遊戲開始前，GM 重新整理絞肉機表。

:::note[城市是社交場所]
城市中有數不盡的派系、社交俱樂部、階層體系、王室、無政府主義集體、運動隊伍、神秘教團、地下圈子、犯罪集團等。他們想要公會擁有的：寶藏、專業技能，以及對自我保全的漠然。公會則想要這些組織所擁有的：影響力、權力、財富……當然，還有華麗的衣著、有用的傳言、有趣的朋友，以及酷炫的據點等。
:::

## 1. 死亡與稅金

當你穿過地下世界的其中一道城門返回時，黑甲全衛軍——服侍城主的精銳戰士——要求徵收你公會流動資產的 50% 作為公會稅。

流動資產意指「金幣」。地下世界充滿著印有五芒星的金幣。

非流動資產是藝術品與奢侈品，例如畫作、雕像、寶石（未切割與切割過的）、書籍、裝飾性武器、珠寶等，全衛軍不會沒收這些。

這項稅收是城市為了支付冒險者公會頻繁引發的災難而設立的。稅收進入一個基金，用於支付變成火災的酒館打架賠償、被當嬰兒帶回地面的膨脹黏液善後費，以及全衛軍的薪水（以確保他們不受賄賂）。

:::note[老天爺！我是主權公民！]
說實話，50% 的稅率最初只是在試玩中一時興起加入的。然而，它在試玩者中引發了如此強烈的反應，顯然必須保留在核心遊戲中。儘管所有人只是在紙上移動著虛構的數字，玩家們卻非常熱衷於逃稅。而且，在某種有意義的層面上，稅收讓返回城市這件事變得令人不快。城市階段應該是稀少的。高稅率意味著玩家將返回地面視為最後手段。
:::

:::note[歷史注腳]
你知道「四位雅士」樂隊唱的歌〈伊斯坦堡〉嗎（後來由「他們可能是巨人」樂隊翻唱而聲名大噪）？為什麼這座城市叫做伊斯坦堡，這個問題有個答案：它源自希臘文 εις την Πόλιν（istanbulin），意思就是「城市」。君士坦丁堡曾是古代世界的中心，你甚至不需要用正式名稱稱呼它，它就是「那座城市」。

《蠕蟲陛下》中這座城市的名字，也僅僅是「城市」，這是它的通稱（雖然未必是正式名稱）。它在過去曾有其他名字（就像君士坦丁堡也曾叫做拜占庭、新羅馬、奧古斯都安托尼亞等），但如今人們就叫它「城市」。城市過去有哪些名字？現在的政府是否一直掌握政權？曾經有哪些古老的力量居住於此？這些都由你們的小組來決定。
:::

## 2. 卓越事蹟

當你返回城市時，抹去公會名冊上最舊的一項卓越事蹟。接著，回顧自上次城市階段以來，公會是否做了任何值得記錄的事。桌邊所有人都可以提出幾件可能稱得上卓越事蹟的事情，例如擊殺地城主、在地下世界開闢新路線、摧毀一處顯赫建築，或奪回一件強大的聖物。GM 裁定什麼算是真正卓越的事蹟。

公會的聲望等於目前有效的卓越事蹟數量。你的聲望影響 GM 角色對待你的方式：越著名的冒險者獲得越多的關注，無論是好是壞。

公會同時最多可擁有五項有效的卓越事蹟。若超過五項，選擇其中最值得讚揚的五項保留。聲望與卓越事蹟的完整說明見第 46 頁。

若你返回城市卻自上次到訪以來一事無成，抹去名冊上最舊的事蹟，並相應地損失 1 點聲望。

## 3. 城市事件

城市從不沉悶。當你搖搖晃晃地回到地面，便踏入這座充滿陰謀、政治與奇觀的混亂大都會。

在遊戲開始前，GM 應建立一張城市事件表，列出框架公會到訪的潛在事件。在此步驟，GM 抽取一張大阿爾卡那牌，查閱此表並解讀結果。

結果不必即刻發生。例如，若 GM 在城市事件表上抽到傳聞，他們可以讓委託人或商人在城市階段的流程中向公會傳遞這則傳聞。

城市事件表應有二十一個條目，遵循以下公式：

**奇遇（I–V）：** 奇遇是偶然的驚喜時刻。GM 們，用這些事件為遊戲注入色彩。奇遇創造一種真實感，是展示你遊戲中城市風貌的機會。奇遇事件是黑暗預兆（如狂人在血月下高呼懺悔），還是田園風情（如彭趣和茱蒂的木偶戲）？

**大事（VI–X）：** 大事是勾勒城市居民生活的事件，包括農神節、盛宴與聖日等打破年輪的節慶。大事應讓玩家感受到每次城市階段的不同。在宗教節慶期間到訪城市，與在平淡日常中造訪截然不同。

**傳聞（XI–XV）：** 傳聞暗示地下世界的內容：寶藏、怪物、陷阱、危險與歷史。傳聞是講述學識、定義地城「故事」的機會。

**旅途事件（XVI–XX）：** 旅途事件往往是前往城市途中出了差錯的事情，或迫使玩家做出艱難抉擇的機會。

**預兆與徵象（XXI）：** 預兆與徵象是特殊事件，地下世界的怪異之物滲出並使城市的生活複雜化。抽到此結果時，GM 查看小阿爾卡那棄牌堆頂端的牌，然後查閱預兆與徵象表（第 153 頁）。

:::note[]
如同絞肉機表，GM 應保持城市事件表整潔且常換常新，用新內容替換已使用的條目（見第 160 頁步驟 8）。下一頁提供了一張範例城市事件表作為起點。
:::

| VII. | **蘑菇森林：** 帶著粉紅色菌褶的黃色蘑菇從一個隨機街區的石板路縫隙中噴湧而出，高如立柱，阻斷了交通與商業。本次城市階段，該街區的特殊城市行動無法使用。 |
| VIII. | **血紅之星：** 一顆彗星懸掛在城市上空，在白晝與黑夜中如同一抹月事般橫貫天際，其病態的紅棕色光芒使城市中的人們難以入眠。你所遇到的每個人都憔悴不堪、眼神呆滯，幾乎維持不住神志。你也無法入睡——帶著壓力狀態開始下一個探索階段。 |
| IX. | **水中之物：** 漁夫從灰河的水中打撈出某個龐然白色生物的屍體。接下來一週，海洋歌唱著一首令人心碎的哀歌，那歌聲引發一種令人難以承受的百無聊賴之感。幾個 GM 命名角色走入海洋，再也未曾現身。 |
| X. | **狗隻集體自盡：** 城市中所有的狗在一夜之間集體自盡：牠們從橋上躍下，撞穿玻璃窗，在灰河中溺斃。 |
| XI. | **雕像的黑色奇蹟：** 城市中所有的偶像雕刻開始說出可怕的褻瀆之詞。在一陣破壞偶像的狂熱中，教團開始搗毀所有具象藝術。雕像、畫作、神像、馬賽克、船首像、兒童玩具等，全都成為其聖火的燃料。近期宗教狂熱大幅升溫。 |
| XII. | **蜜糖教團：** 一名冒險者將黑蜜帶回了城市。凡食之者都聲稱看到一個懷孕女人的可怕而美妙的異象，他們狂呼著「初生大軍」。他們的瘋言瘋語莫名地在你心中引發深深的焦慮。 |
| XIII. | **沉默之子：** 一個巨大的嬰兒沿著街道爬行，周身籠罩著一股無法穿透的寂靜：在牠的存在中，任何聲音都無法傳遞。牠似乎在追獵公會。公會可以躲藏以延緩這場厄運，但沉默之子若不受約束，必將造成某種可怕的傷害。 |
| XIV. | **死而不死：** 死者縈繞城市。在公會找到地下世界不死軍團的源頭並加以平息之前，無法執行任何城市行動。若任由這場死者瘟疫蔓延，地下世界將突破至地面，吞噬城市（正如它曾吞噬許多城市一樣）。 |

## 4. 繳交委託

繳交在探索階段完成的任何委託任務。拜訪你的委託人，報告你的成功（或失敗），並領取應得的酬勞。這通常能給你支付生活費所需的金幣。

將委託任務繳交後，每完成一個委託給予 1 點經驗值。若沒有完成任何委託，跳過此步驟。

在此步驟中，冒險者可以出售藝術品與奢侈品，例如寶石、畫作和珠寶。GM 在建構地下世界時確定所售物品的價值；詳見第 182 頁「寶藏有多少？」。賣家在支付生活費之前先收取款項。

## 5. 生活費

城市的生活並不免費。生活費支付你的飲食和住宿，並補充你的背包裡的探險裝備。

裝備費用不逐一計算。支付某個生活費等級，即可帶走你能攜帶的該等級（或更低等級）的裝備。探險裝備清單詳見第 162 頁的臍帶市集。

生活費共有四個等級：赤貧、貧困、普通與奢華。

**赤貧—0 金**

若你無力（或拒絕）支付生活費，你將睡在街頭。狗舔你的傷口。你沒有東西吃，並帶著壓力狀態開始下一次冒險。你無法購買任何新裝備。

**貧困—25 金**

若你一貧如洗，你睡在濟貧院或廉價旅店的大通鋪。你既不獲得特殊好處，也不受到特殊懲罰。

你可以購買盡可能多的貧困物品清單裝備。

**普通—50 金**

普通住宿讓你入住一家不錯的旅店。在此步驟，你可以消耗充能羈絆治癒傷勢並恢復決心，如同紮營階段步驟 4（復原）（見第 141 頁）。

你可以購買盡可能多的貧困和普通物品清單裝備。

**奢華—100 金**

若你住在最高級的場所享受奢靡生活，你將完全治癒所有傷勢，並以完整的決心開始探索。

你可以購買盡可能多的貧困、普通和奢華物品清單裝備。
