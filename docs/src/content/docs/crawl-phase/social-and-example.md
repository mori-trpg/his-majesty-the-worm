## 社交遭遇與立場

在地下世界中，你將遭遇地城的奇異住民，以及那些竭力試圖消滅他們（或消滅你）的冒險者。

角色扮演你遭遇的角色是 GM 眾多職責之一。GM 被提供了稱為立場的角色扮演指導方針。GM 創建具有有趣個性的角色並主持社交遭遇的建議[主持人指南](/gamemastering/)。

每種立場都有三個嚴重程度：輕微、基礎，以及激烈。輕微情緒位於圓圈內側；激烈情緒位於圓圈外側。相關情緒彼此相鄰分組。

### 起始立場

角色的起始立場告知 GM 該生物最初應如何行動、它對公會的敵意或友好程度，以及他們在談判中可能的訴求。

有時，角色的起始立場會根據公會遭遇它的情境而顯而易見。若有任何疑問，或若希望引入隨機性，GM 應查看小阿爾卡那棄牌堆頂部的牌。

| 牌 | 立場 |
|---|---|
| I–II | 憤怒 |
| III–IV | 厭惡 |
| V–VI | 悲傷 |
| VII–VIII | 喜悅 |
| IX–X | 驚訝 |
| 侍者–騎士 | 信任 |
| 皇后–國王 | 恐懼 |

### 立場的影響

具有正面立場（例如喜悅、信任）的角色會誠信地進行談判。他們會在接受公會合理交換的情況下順應合理的請求。擁有激烈正面立場的角色甚至可能提供額外的援助和建議。

具有負面立場（例如憤怒、厭惡）的角色希望盡快結束對話。他們對與公會交換商品或服務沒有任何興趣。若遭遇繼續，將以對抗的方式進行——轉換至挑戰階段。

當你與 GM 的角色進行角色扮演時，他們的立場會根據你的行動與其個性的互動而變化。所有智能生物都可以進行協商（前提是有共通語言）。與 GM 的角色進行角色扮演時，你不必說愚蠢的腔調或使用偽莎士比亞式語言，甚至不必逐字說出你的角色在說什麼。但你必須說出你想要什麼，以及你準備如何推進對話。

- 你可以奉承虛榮的女伯爵。

- 你可以向諂媚的牧師說出虔誠的陳腔濫調。

- 你可以向哥布林廢品商提供稀有的廢品。

這些策略只有在針對特定角色時才有意義；向虛榮的女伯爵提供廢品是不明智的。

GM 根據你所說的內容如何與其角色個性契合或衝突，裁定協商的結果。根據角色扮演交流的結果，GM 角色的立場可能會在嚴重程度上增加或降低，或轉移至相關的情緒。

- 能夠吸引角色喜好與需求的好主意和有力籌碼，將自動創造正面立場。

- 與角色厭惡之事相衝突的方式，將自動創造負面立場。

- 當情況危險且有些東西岌岌可危時，GM 將要求進行權杖考驗。

- 此考驗成功，將角色的立場移至相鄰的、有利的情緒，或提升當前情緒的嚴重程度。

- 失敗，則將角色的立場移至負面情緒。

## 角色扮演立場

共有七種基本立場：憤怒、厭惡、悲傷、喜悅、驚訝、信任和恐懼。以下列出了七種立場，每種立場的核心情感狀態側翼為其輕微和激烈狀態。同時列出了情感同義詞，其中某個詞語可能更為貼切，有助於 GM 適切地扮演該立場。此外還提供了一些關於具有該立場的生物在挑戰中可能如何行動的注記。

### 信任（接受→欽佩）

信任是對對方整體誠實性、公平性或能力的信念衡量。基本程度的信任是許多善意生物常見的起始立場。信任公會的生物通常會誠信地進行談判，但仍會謹慎地確保交換公平。背叛一個角色的信任，是將其立場改變為悲傷或憤怒的好方法。

同義詞：自信的、充滿希望的、樂觀的、好玩的、有興趣的、好奇的、探究的

在挑戰中：通常，具有此立場的生物不傾向戰鬥。若情況在不改變信任立場的前提下迫使戰鬥，該生物可能試圖通過非致命的方式，進一步了解公會的實力。

### 悲傷（憂鬱→悲慟）

悲傷是與失落、絕望、無助或失望相關的情感痛苦。悲傷的生物通常有明顯需要被緩解的需求，這可能使他們的立場變得更好。然而，悲傷常常使生物對他們曾經享受的活動失去興趣，悲傷會為正面影響設立重大障礙。

同義詞：焦慮的、空虛的、無助的、絕望的、沮喪的、易怒的、愧疚的、孤獨的、被遺棄的

在挑戰中：在戰鬥中，深陷悲傷的生物會奮不顧身地戰鬥。他們會冒著他們平時可能不會冒的風險。通常，他們會試圖在先攻中搶先行動，魯莽地衝鋒入戰並將自己暴露於危險中。

### 憤怒（惱怒→暴怒）

憤怒是一種由感知到的挑釁、威脅或傷害所引發的強烈情感反應。憤怒的表現極具攻擊性和威嚇性，可能迅速演變為暴力。這是具有獸性智慧的生物或社會適應不良的人類的常見情感反應。憤怒是戰或逃反應的一半。憤怒可能損害生物在遭遇中接收信息或做出明智決策的能力。

同義詞：受傷的、被羞辱的、受到威脅的、充滿仇恨的、受到侵犯的、狂怒的、被激怒的、充滿敵意的、被嘲笑的

在挑戰中：憤怒使人愚蠢而笨拙。憤怒的生物將試圖集中攻擊其憤怒的根源。憤怒的生物將不顧環境地魯莽攻擊。

### 厭惡（無聊→憎惡）

厭惡是對令人反感、不快或不愉快的事物的反應。這種反感通常是打破衛生規範或道德準則的產物。這是你遭遇的潛在對抗性生物的常見起始立場。記住，冒險生活是骯髒、骯髒的生意。

同義詞：不贊同的、被排斥的、評判的、震驚的、反胃的、恐懼的、猶豫的

在挑戰中：充滿憎惡的生物以對其厭惡目標施加暴力為樂。該生物的主要慾望是造成傷害或痛苦。這種傷害可能針對特定的生物類型（一名追逐女巫的捕巫者），或者可能更普遍（吸血鬼對所有凡人生命的普遍蔑視）。

### 恐懼（不安→驚恐）

恐懼是對感知到的危險或威脅的情感反應。恐懼是戰或逃反應的另一半。以恐懼回應的生物通常只想結束這次社交遭遇，可能願意做出任何必要的讓步來達成這一目的。或者，若他們認為可以成功，受到驚嚇的生物會試圖完全放棄這次遭遇並逃離。

同義詞：脆弱的、被拒絕的、受到威脅的、暴露的、緊張的、受迫害的、自卑的、擔憂的、無助的

在挑戰中：恐懼的生物會試圖逃離其恐懼的來源，除非有必要逃脫，否則避免戰鬥。他們優先使用高價值的牌作為先攻。

### 驚訝（分心→敬畏）

當期望與現實之間出現落差時，生物會以驚訝作出反應。驚訝可能出現在公會與城市更「正常」的社會互動時——並非每個人都習慣於冒險者的生活。驚訝也可能在地下世界中令人困惑或震驚的遭遇中出現。

同義詞：被嚇到的、困惑的、驚嘆的、震驚的、沮喪的、困惑不解的、興奮的

在挑戰中：一個驚訝的生物今天並未計畫要戰鬥。若它認為自己受到威脅，它會戰鬥，但不會出於自身目的主動尋求戰鬥。它會盡早試圖脫離戰鬥。

### 喜悅（滿足→狂喜）

當需求得到滿足時，生物會感受到喜悅或幸福感。與憤怒一樣，喜悅的情感也可能遮蔽判斷力。這是影響生物的理想狀態，因為它們對懇求和交換具有良好的意願。

同義詞：快樂的、平靜的、親密的、亢奮的、自豪的、感激的、充滿力量的、受到尊重的

在挑戰中：喜悅的生物通常不渴望投入挑戰。然而，若情況需要，喜悅的生物可能會自大或過於大膽，將戰鬥視為一場遊戲。自大的生物可能在戰鬥中與冒險者嬉耍、嘲弄或挑釁。
