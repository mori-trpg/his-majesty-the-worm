探索階段的流程

《蠕蟲之王》的新遊戲從通往地下世界的大門開始。探索階段始於公會踏入那無光深淵的第一步。

探索是遊戲中以探索地下世界為核心的階段。當公會離開文明的安全地帶去追尋任務，與奇異生物周旋、躲避陷阱、挖掘古老寶藏時，他們便是在探索。遊戲的大部分時間都集中於此階段。

地下世界是所謂的巨型地城。巨型地城本質上是奇異的。它不需要符合玩家的觀念與期待，但從中保有內在邏輯是有益的。學習地下世界的「規則」是有趣的，而具備內在一致的邏輯能讓玩家做出有意義的選擇（這正是遊戲的目的所在）。

巨型地城由較小的構成地城組合而成。每個地城都是一個主題性的層級、區塊或子區塊。例如，你的地下世界可能由五個地城層級組成，分別稱為尖塔、命運交錯之堡、倒置城堡、廢墟之城與奧特亡城。

地城被分割成不同的房間。一個房間可以是任何獨立的空間：一個廳室、一個宴會大廳、一個天然洞穴、一條地下河流，等等。每個房間都應包含一些有趣的東西，供玩家互動。例如，命運交錯之堡中的某個房間可能有一座以巨人骸骨堆砌的巨大王座，另一個房間可能是一隻需要聆聽謎語才能存活的反向斯芬克斯的巢穴，諸如此類。

當公會探索每個房間時，稱為絞肉機的隨機事件會製造緊張感。絞肉機代表時間的流逝：火炬燃燒殆盡、巡邏的怪物，以及其他危險。

每個地城層級都有一份地圖和一張絞肉機表，兩者都充滿了有趣的發現。當玩家進入一個新房間時，GM 查看房間描述，同時從大阿爾卡那抽取一張隨機牌，並對照絞肉機表。然後他們在隨機絞肉機遭遇的背景下描述新進入的房間。

本章詳述玩家在探索階段需要了解的內容。GM 主持探索階段的相關建議見第 178 頁。GM 建構地下世界的程序見附錄 E：地下世界建構，第 330 頁。

神話地下世界
《蠕蟲之王》的設定是一個在小說、民間傳說與遊戲中被描述為神話地下世界的地方。神話地下世界並非僅僅是一個在每個陰暗角落都藏著哥布林和食人妖的洞穴系統。它是另一個世界，充滿了非常規的物理法則、非人類的存在，以及異域的建築。

神話地下世界的特徵

1. 它很大，有許多層級；事實上，它可能是無盡的
2. 它遵循自己的生態與物理規則
3. 它並非靜態的；其中的居民，乃至整體布局，都可能隨時間生長或改變
4. 它並非線性的
5. 有許多方式可在各層級之間上下移動
6. 其目的神秘莫測，或籠罩在傳說之中
7. 它對探索者充滿敵意
8. 越深或越遠的層級越危險
9. 它是戰役的核心（或說，唯一的）特色

——《地城作為神話地下世界》，菲洛托米沉思錄

遊戲通過 GM 的描述與玩家的決策組合來進行。這並非一個預先設定好的故事。

在探索過程中，玩家告訴 GM 他們想要去哪裡。他們使用一張角色內的地圖，以確保所有人都清楚公會目前的位置。

GM 描述公會在旅途中遭遇的一切。GM 在隨機絞肉機事件的背景下描述每個房間的內容。

當公會遭遇危險——壓縮的牆壁、牛頭人迷宮守護者、反生命力場、欺詐的商人——玩家與 GM 展開對話。

• GM 描述情況，玩家詢問關於場景的問題，GM 予以澄清。

• 玩家描述他們的冒險者試圖做什麼，GM 詢問關於他們行動與意圖的問題，玩家予以澄清。

• GM 描述冒險者行動所產生的結果。若冒險者完成目標的能力存在風險或疑問，GM 要求進行命運考驗（第 8 頁）。

「人們總喜歡假裝自己了解這個世界的一切。[……]你的地圖上沒有空白，所以繪製地圖的人在某片空白上加了陰影，寫上『艾爾德之地』。就這份用處而言，你還不如直接在地圖上燒個洞。」

——《智者之恐懼》，派翠克·羅斯弗斯

儘管地下世界中存在著超自然和奇異的存在，幾乎沒有生物會一見就攻擊並戰鬥至死。（嗯，也許骷髏除外。他們想要把你殘忍地囚禁著的骷髏兄弟們從肉身監獄中解放出來。）幾乎所有事物都可以交談、協商，或至少加以躲避。關於與 GM 的角色進行協商的概覽，見第 99 頁的「立場」。

當戰鬥無法避免時，探索階段轉換為挑戰階段。一旦挑戰解決，挑戰階段便轉換回探索階段。

神秘地，地下世界具有一種維持生命的特性。當一擊足以致命時，受害者會在垂死邊緣徘徊，而非立即死亡。這一特性延長了許多遭受地下世界陷害的冒險者的生命。

而那些確實在地下世界中死亡的人，將作為不死生物再度復甦……

謠言磨坊
《蠕蟲之王》的一大重點在於探索與發現。若一個公會的冒險者全部（或者，上天保佑，是所有人）死亡，重新發現並重走舊路是乏味的。這就是我們設有謠言磨坊規則的原因。

冒險者在酒館中永遠在談論他們發現了什麼、與什麼戰鬥過、以及什麼差點把他們整個吞掉。除非冒險者特意保守某些探索內容的秘密，否則可以假設這些都是大眾普遍知曉的信息。因此，之後加入遊戲故事的冒險者可以知道前任冒險者所發現的事情。他們知道秘密開關在哪裡，他們知道如何擊敗蛆母，他們知道如何安全地穿越那座隱形橋梁。

凡入此門，棄絕一切希望

隨著你探索，你將發現需要繞過、躲避或撤退的危險。

這是一個將世界視為真實存在來互動的遊戲。因為角色扮演遊戲是對話型遊戲，你通過提問來探索遊戲世界。你可以詢問你的冒險者能看見什麼、感受到什麼、聽到什麼，等等。無需抽牌——只需詢問你能感知到什麼，GM 就會如實回答。在 GM 描述事物時，通過具體的追問深入探索遊戲世界。

沒有技能或能力的清單來限制你在虛構世界中能做什麼。不要問「我能進行某種考驗來尋找……」或「我能進行考驗來拆除……」。而是改問「當我緩緩打開門時，我感覺到任何張力嗎？」或「如果我敲擊這面牆，我能聽到任何聲音嗎？」通過提問審視你的周圍環境，並說明你的角色的行動：GM 會告訴你是否需要進行命運考驗。

話雖如此，你的角色卡上可能有些東西能讓你佔有優勢！你可能有一件物品、一個法術或一個天賦，可以巧妙地加以利用。你的成功與失敗，取決於你的想法。

鎖門
有些門是上鎖的。

若門上有鑰匙孔，你可能會在這個層級的地城居民身上的口袋裡找到鑰匙。鑰匙通常有對應的編碼：銅鑰匙配銅門，美人魚形狀的鑰匙配有美人魚圖案的門，以此類推。

若門上有鑰匙孔，你也可以嘗試使用開鎖工具撬鎖。使用開鎖工具撬鎖需要以錢幣成功通過命運考驗。

你也可以嘗試強行破門。你可以用寶劍考驗將其撞開。像撞門錘或撬棍這類工具可為此考驗提供優勢。

有些門只能通過特定機關才能打開。你需要說出特定通行密語、扭動標本麋鹿頭、踩下房間中的全部四塊壓力板，等等。若一扇門看似卡死，稍後再回來看看。在進一步探索後，你可能對如何打開它有了更多思路。詢問地城居民是否有什麼建議。

暗門
有些門是隱藏的。這些門可能被遮蔽（藏在書架後方），或建造得幾乎難以察覺（只有扭動燭台才能打開的石門）。

你通過描述你的搜索方式來找到暗門。你可以查看一面牆，看看能否發現任何接縫。你可以敲擊牆壁，聆聽是否聽起來是空心的。你可以查看帷幔後方、衣櫃內部、地毯下方，看看是否有任何異狀。

查看你的地圖。有沒有哪個空白位置邏輯上應該有一個房間？在那裡尋找暗門。

尋找規律。暗門藏在對它們而言合情合理的地方。若你在一幅畫像後方找到一扇暗門，就搜索其他畫像後方，看看是否有某種規律可循。

若你沒有詢問具體信息，只是說「我花盡所需的時間搜索房間裡的暗門」，GM 可以使用第 91 頁的「謹慎移動」規則。他們告訴你是否有暗門，但會從絞肉機抽牌，以確認你在搜索過程中花費的時間是否使火炬熄滅。

搜查陷阱
有些門、房間或通道設有陷阱。你通過描述你的搜索方式來找到陷阱。

非常緩慢地打開門，感受鉸鏈中是否有一個會觸發陷阱的機關。彎腰檢查門框中是否有絆線。小心行走，檢查走廊中是否有任何壓力板。

若你制定了一個程序，例如「我們沿走廊前行，用十英尺長桿在我們前方敲打地面」，由於搜索所需的時間，GM 可以使用第 91 頁的「謹慎移動」規則。他們告訴你是否發現了任何陷阱，但會從絞肉機抽牌，以確認你在搜索過程中花費的時間是否使火炬熄滅。

應對陷阱
有些陷阱無論你搜索得多麼仔細，都難以避免。你決定如何應對你觸發的陷阱。

有些陷阱就此發生——房間在門一打開時便開始注水。其他陷阱允許你進行命運考驗以避免最壞的結果——你以錢幣考驗，在崩塌的橋梁倒塌時抓住橋邊。

觸發陷阱時你描述自己行動的方式，將決定 GM 如何裁定接下來的事情。

以下是一些範例。

範例一
當公會踏入一個房間時，門猛地關上並上鎖。伴隨著研磨聲，東牆和西牆緩緩向彼此靠近，威脅著要像巨大的垃圾壓縮機一樣把所有人碾碎。

你打算怎麼做？

一些建議：

• 有沒有公會成員在門關上時被隔在外面？他們有沒有辦法找到任何控制裝置，來停用這個陷阱？

• 你的背包裡有岩釘嗎？你能把它們打入牆中，以減緩或阻止牆壁的推進嗎？

範例二
你找到一扇嵌在一個形似綠色惡魔臉孔的門框裡的門。綠色惡魔的嘴裡是一個黑洞。不只是黑暗，一個字面意義上的虛空存在於綠色惡魔的嘴裡。

你打算怎麼做？

一些建議：

• 謹慎是勇敢的最佳部分。若你不需要應對綠色惡魔臉，你能就這樣離開嗎？

• 若你需要應對這張臉，附近有任何隱藏的開關或按鈕嗎？你能按壓它的眼睛嗎？你能扭動它的角嗎？

• 不要用裸手觸碰嘴裡的虛空。向裡面投入一顆彈弓石。用一根長桿從遠處觸碰它。

• 你的公會中有人擁有反制法術天賦嗎？它對那張臉有任何效果嗎？

範例三
當你走下走廊時，你踩上了一塊壓力板。你聽到一聲「喀嚓！」，它沉入地板中。你聽到牆壁中有機關的聲音，就像一台機器在啟動。

在需要即刻做出決定的情況下，你掌握的信息不多——但做點什麼總比什麼都不做要好。

你打算怎麼做？

一些建議：

• 你能在壓力板上保持壓力，以防釋放它才是陷阱的觸發條件？

• 你能俯趴在地，試著躲避即將到來的威脅嗎？你能在壓住踏板的同時這樣做嗎？

• 你有盾牌嗎？若陷阱被觸發，你能舉盾保護自己免受潛在威脅嗎？

• 你有黏液炸彈嗎？你能投擲它來摧毀部分牆壁或門嗎？

• 你的公會中有人擁有神盾天賦嗎？若陷阱被觸發，他們能為你的利益使用它嗎？

• 你有攜帶式虛空洞法術嗎？你能將它施放在牆壁上以看見背後的機關嗎？若你將手伸入洞中，你能卡住或破壞那個機關嗎？

• 門與牆齊平嗎？門框中有任何可以藏身的空間嗎？

• 你能搜索是否有暗門嗎？牆上有沒有維修面板？天花板上有嗎？地板上有嗎？

• 你有撬棍嗎？你能撬開門嗎？

• 你的隊伍中有沒有巨魔？他們擁有巨體天賦嗎？他們能憑藉蠻力撞開門或破壞牆壁背後的機關嗎？

地圖

在遊戲前，GM 會繪製地下世界的地圖，並為每個房間的內容建立詳細的筆記。他們也會為玩家製作一份每個地城層級的基本地圖：這就是你的公會的角色內地圖。

你的地圖讓你告訴 GM 你的公會想要去哪裡。例如，你可能說：「我們要前往 202 號房間。」由於 GM 的筆記與玩家地圖上的房間編號相同，GM 可以翻到他們對 202 號房間的描述，並以最少的頁面翻閱繼續遊戲。

玩家一側的地圖不包含任何關於地城房間內容的信息。然而，每份地圖都應：

• 顯示房間的基本輪廓；

• 顯示穿越地城的路徑；以及

• 附有鍵值，每個房間有不同的編號。

雖然地圖提供了一些信息，但它可能並不完整！當你旅行時，你可能發現可以直接標注在地圖上的隱藏房間。探索與發現秘密，正是遊戲的樂趣所在！

當你在地圖上填寫關於你遭遇的房間、陷阱和地城居民的筆記時，你將能夠在探索過程中做出明智的決策。

時段

旅行時間以時段來衡量。一個時段並非一個具體的時間段；它可能是一個小時、十分鐘，或整夜。在地下世界中，很難分辨實際上過了多少時間。時間以奇異的方式流逝——快樂的時刻轉瞬即逝，而緊張的時刻則像拉長的陰影般延伸。時段只是一種談論你移動速度快慢、安靜程度，以及謹慎程度的方式。

一個時段在以下情況過去：

• 每當公會進入地圖上的一個新房間時；

• 每當公會花費很長時間執行一項任務（例如將門從鉸鏈上取下、修理損壞的機器、數清一罐中的軟糖豆）；以及

• 每個紮營階段和挑戰階段。

當一個時段過去時，GM 從大阿爾卡那牌堆中抽取一張牌，並對照該地城的絞肉機表比較結果。GM 隨後在新的或當前場景的背景下描述絞肉機事件。

絞肉機

絞肉機是一系列短暫事件的集合——諸如怪物巡邏、天氣，或與競爭公會相遇。重要的是，你的光源閃爍並熄滅的速率由絞肉機控制；關於光源的更多信息見第 94 頁。

當你旅行時，GM 將描述結合每個房間內容與絞肉機結果的場景。例如，若你進入的房間是一個龍的巢穴，而 GM 從絞肉機表中隨機抽取了一隊哥布林巡邏，你可能會發現哥布林們正試圖潛入龍的巢穴竊取財寶——並在此過程中製造了相當大的噪音。

當一個絞肉機事件被觸發時，GM 應將其標記掉。若牌被洗牌後 GM 再次抽到同一事件，則什麼都不會發生，公會將享有一個時段的喘息。GM 在每個城市階段後補充他們的絞肉機表。

以下是絞肉機表的基本結構：

• 火炬熄滅（I–V）：火炬熄滅代表時間流逝，你的光源逐漸黯淡。當火炬熄滅時，所有光源都會搖曳。

• 火炬熄滅事件不會從絞肉機中標記掉。它們可以多次發生。

• 奇象（VI–X）：奇象是緣分巧合的時刻。「什麼都沒發生」對於隨機表來說是一個無聊的結果。奇象是 GM 為地下世界的黑暗深處注入色彩的機會。它們可以為角色扮演交流提供靈感，或提供關於本地環境的線索。

• 旅途事件（XI–XV）：旅途事件代表旅途中出錯的事情。它們可能要求你就資源管理做出艱難抉擇，或要求你進行命運考驗以避免困境。旅途事件可能引發挑戰。

• 隨機遭遇（XVI–XX）：隨機遭遇是與地下世界居民的偶然相遇。這些可能是與潛在盟友的友善遭遇、與飢餓動物的危險遭遇，或與怪物的致命遭遇。有趣的隨機遭遇不僅僅列出「怪物類型」。最好的隨機遭遇描繪出一個小場景，GM 可以以此為靈感發展成一場挑戰。

• 任務傳言（XXI）：任務傳言讓你更進一步接近當前任務。任務傳言提供關於你的任務目標的提示或方向。若你已在當前任務上取得了顯著進展，這個結果甚至可能出現任務的目標本身。

為地城層級建立絞肉機的完整程序詳述於附錄 E，第 340 頁。以下是基本內容。

大噪音

一般來說，每當公會發出巨大噪音或以其他方式引起注意時，GM 應拉動絞肉機表。若抽牌結果為隨機遭遇（結果 XVI–XX），GM 描述一次與被噪音吸引來的遊蕩生物的遭遇。否則，什麼都不會發生。

謹慎移動

若公會移動得非常緩慢或有條有理，在他們前方用十英尺長桿敲打，GM 應在解決潛在遭遇後再次拉動絞肉機表。若第二次抽牌結果為火炬熄滅（結果 I–V），GM 保留該結果，以代表額外時間的流逝。否則，什麼都不會發生。
