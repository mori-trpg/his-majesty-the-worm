## GM 宣言

遊戲的核心在於決策。無論是傳統遊戲如撲克，或是桌遊，還是角色扮演遊戲，本質上皆如此。

角色扮演遊戲的遊玩透過對話進行。GM 描述場景，玩家提出問題，GM 進一步說明，玩家描述行動，GM 再度提問，玩家繼續補充，如此循環往復。這是一種來回互動——提出問題並回答問題。在此交流中，GM 與玩家共同做出決策。這正是角色扮演遊戲的樂趣所在。

你的職責，在於為玩家創造引人入勝的決策。以下是一些建議：

### 創造引人入勝的角色

地下世界和城市都應充滿擁有獨特有趣動機的難忘角色。GM 筆下的每個角色，都應擁有由喜好、厭惡、慾望和需求所定義的鮮明個性。

- 創造能引發有趣決策的環境
- 創造擁有有趣動機的角色與派系

如同應預先廣播危險，你也應清楚廣播角色的個性。藉由清楚呈現角色的喜好與厭惡，玩家便有機會進行有意義的角色扮演。

### 創造有意義的探索

角色扮演遊戲中的決策應該是有趣的，而非武斷的。在探索階段中，提供有意義的選擇至關重要，使探索既充滿樂趣又引人入勝。

一條有兩個分叉路口的空曠走廊，不能給玩家足夠的資訊做出明智或有趣的選擇。擲硬幣就能決定「最佳」路線。然而，若你在右側通道的牆上寫上黑話塗鴉：「前有不死生物！勿入！速退！一切皆失！」若玩家側耳傾聽左側通道，他們可以聽到雞蛇那種獨特的、孔雀般的啼叫聲。如今玩家便有了做出決策的資訊：他們更願意面對雞蛇還是不死生物？他們更有把握應對哪一個？他們能否將不死生物引向雞蛇？

預先廣播危險。將寶藏藏在能讓玩家感到自己聰明的線索之後。創造有意義的選擇，讓玩家能做出明智的決策。

地下世界雖然危機四伏，但並非每個居民都必須是敵對的。立場規則（[社交遭遇](/crawl-phase/social-and-example/)）確保玩家在遭遇新角色時有公平的機會。給予玩家收集資訊、結交不可思議的盟友，以及參與派系鬥爭的機會。

- 充分描述環境，調動視覺、聽覺、嗅覺、味覺和觸覺
- 以誠意行動，慷慨地分享資訊
- 公正裁決玩家行動

### 調動感官

有意義的探索，只有在玩家對環境有充分感知的情況下才能奏效。玩家需要你盡可能多地告知他們所在環境的一切。用描述繪製圖景，當腦海中的畫面難以構建時，實際勾勒出房間的草圖，並生動描繪五感。不要遺漏任何玩家能夠看見、聽到、感受或聞到的重要細節。

### 慷慨發言

不要落入認為半藏的秘密和暗示是有趣的陷阱。資訊分享遊戲只有在資訊真正得到分享時才有趣。也許資訊伴隨著代價（「你不確定那條龍是真的在睡覺，還是只是在裝睡。你需要靠近一點才能仔細看看牠……」），但在需要付出代價的時候，盡你所能給玩家最多的資訊。「揭穿」的瞬間並不好玩。

當玩家投注學識時，這一點尤為重要。若你接受學識投注，你便有社交義務慷慨地回應玩家，給予盡可能多的適當資訊。

:::note[裁決規則爭議]
遊玩中出現規則爭議時，應如何處理？GM 的職責之一是平息這些問題並裁定規則的適用方式。若你還在學習遊戲，你可能需要每次都查閱規則，直到敘事與機制的流動感覺自然為止。若你對規則相當熟悉但遇到瓶頸，我們建議：傾向於對玩家有利的裁決，事後再查閱規則。不要讓翻書的動作打斷一場精彩的戰鬥。暫時制定一條對玩家有利的臨時規則，並請桌邊最熟悉規則的玩家在戰鬥結束後查閱。
:::

### 公平裁決

讓玩家的決策具有分量。若玩家想出了一個應該奏效的方法，你甚至不需要觸碰牌組。他們的計畫可以直接生效。符合情理的事，優先於規則。

同時，也不要手下留情。勝利只有在失敗真實可能的情況下才有價值。你既不是玩家的對手，也不是他們的朋友。你的職責之一，是成為虛構世界物理法則的中立詮釋者。

存在隨機元素。明智地詮釋它們。盡可能讓機制清晰可見。說出你在做什麼、玩家正在為何進行命運考驗，以及結果將會如何。

作為 GM，你可以給玩家帶來你自己也不知道如何解決的困境。玩家將用他們的智慧和解決問題的能力令你驚喜。當他們想出了一個令人驚訝且應該奏效的絕妙方法時，就讓它奏效。

> 「深藏於人類潛意識中，有一種對合乎邏輯之宇宙的普遍渴求——一個有意義的宇宙。然而真實的宇宙，永遠比邏輯高出一步。」
>
> ——《沙丘》，法蘭克・赫伯特

## 主持探索階段

探索階段的標準流程如下：

玩家宣告他們打算前往的方向。若公會在本階段從城市出發，他們宣告將使用哪個地下世界的入口。若公會已在地下世界中，他們宣告移動的方向（例如：「我們要往東邊的第 12 號房間走。」）

- GM 從大阿爾卡那中抽一張牌，用於絞肉機（[探索階段](/crawl-phase/)）。
- GM 詢問：「你們有什麼光源？」
- GM 詢問：「你們要做什麼？」
- 玩家向 GM 提出關於環境的問題。
- 玩家描述他們的行動。
- 為協助裁決，GM 詢問：「你手上拿著什麼？」

根據玩家能夠看到的內容，GM 除了絞肉機的結果之外，還描述房間的內容。

GM 裁決玩家行動的結果。當局勢緊張且失敗具有意義時，GM 要求進行命運考驗。整個過程不斷重複。

### 建構你的地下世界

地城的地圖繪製、怪物數值設計、陷阱的設置，以及獎勵的隱藏，都由你全權負責。《蠕蟲陛下》要求你提前完成這些工作。在開始主持遊戲之前，你需要坐下來，勾勒出你將使用的整個地下世界巨型地城的全貌。附錄 E 涵蓋生成你的地下世界巨型地城的相關說明。

在一兩個週日下午完成這些工作，能讓你從每週的遊戲準備中解放出來。艱難的工作已提前完成。城市階段將是你保持地城新鮮感的觸發點：這是你思考玩家如何改變地下世界局勢的時機，相應地更新你的筆記，並創造新的絞肉機條目以取代玩家已遭遇過的內容。

### 創造音效環境

你可以為地下世界的不同地城建立一份專屬播放清單，提供一致的氛圍和主題。將每個地城層級與一首特定的歌曲關聯。當玩家進入一個地城時，循環播放其主題曲。（選擇帶有循環功能的音樂播放器很重要，Spotify 效果不錯。）

你也可以為每場遊戲的開始和結束、挑戰階段，以及反覆出現的 GM 角色設定主題音樂。

電子遊戲音樂在這方面效果出色。它設計用來融入背景、營造氛圍。但請注意，你的玩家可能會將音樂與特定電子遊戲產生強烈的聯想，從而打破《蠕蟲陛下》的沉浸感。請選擇來自冷門遊戲，或你知道桌邊玩家沒有過多遊玩過的遊戲的音樂。

你的狀況可能因人而異。有些玩家認為任何音樂——即使是沒有歌詞的音樂——都非常令人分心。詢問你的桌友他們的偏好。

> 「因此，你們每個人，當輪到你的時候，必須下降到那個普遍的地下居所，培養在黑暗中視物的習慣。」
>
> ——〈洞穴寓言〉，《理想國》，柏拉圖

:::note[自由改造]
主持遊戲有點像烹飪。前幾次你會謹慎地遵循食譜。然而，烹飪次數越多，重複、實驗、即興和限制便越能揭示食譜背後的方法論和化學原理。

這款遊戲是一個完整的套裝，但它只是一份食譜。它有一種能產生特定風味的底層邏輯。你玩得越多，就越能理解各個部分如何相互配合。一旦你理解了，你便能知道如何改變部分內容，以更符合你的口味。當遊戲的主導權不在書本，而在 GM 和桌邊所有人的共同意志時，遊戲才會更加精彩。

若這本書提到某個內容只是一筆帶過，比如說，狼人症，我們沒有必要為你提供大篇大篇的詳細規則。角色扮演遊戲已存在數十年，相關資料俯拾皆是。在遊戲場次之間找些時間，創造一套你喜歡的狼人症規則。讓相關的學識符合你打造的遊戲世界。與玩家溝通這個想法，讓所有人都為此感到興奮。
:::

## 創造有趣的角色

你的職責是刻畫出個性引人、奇特、充滿矛盾，且清晰廣播的角色。無論你控制的生物有多麼可怕，這都是真理！

你介紹的每個角色，都應擁有由厭惡、喜好、慾望和需求所定義的個性。

理解一個角色的個性，能讓你公平地裁決公會與其交涉的嘗試。關於角色扮演的指引，請參考[社交遭遇](/crawl-phase/social-and-example/)的立場說明。

### 慾望

慾望實現角色的心願。

- 慾望是具體的，可以是物品（「足夠用於嫁妝的金錢」）或事件（「與普雷斯科特寡婦成婚」）。
- 若向 GM 角色提供其慾望作為籌碼，試圖影響他們的命運考驗將帶有優勢。

### 厭惡

厭惡觸發負面立場，例如厭煩或憤怒。

- 這些觸發點可以是行為（「說謊」）或特質（「所有精靈」）。
- GM 角色不會與他們厭惡的冒險者進行交涉。

### 喜好

喜好觸發正面立場，例如信任或喜悅。

- 喜好是角色心扉的鑰匙。
- GM 角色會與他們喜歡的冒險者公平交涉。
- 這些觸發點可以是行為（「分享酒精」）或特質（「擅長音樂」）。

舉例來說，有些美人魚可能喜歡八卦，渴望吃掉一個男人的眼睛、指尖和私處。一支由男性組成的公會遭遇美人魚，很可能成為被吞噬的目標。然而，若公會用豐富的八卦餵飽美人魚，她們也許會放棄一場戰鬥遭遇。

### 需求

需求代表維持現狀的根本訴求。

- 需求是抽象而根本的（「讓阿法納斯男爵維持權位，以確保農民繼續向我繳稅」）。
- 若以威脅 GM 角色的需求作為籌碼，試圖影響他們的權杖考驗將帶有優勢，但他們的立場將是敵意性質，例如憤怒或厭煩。他們不會再度進行交涉。

對比或衝突的喜好、厭惡、慾望和需求，確保玩家有多種方式與 GM 角色互動。

## 混沌的恩賜

有時絞肉機事件針對隨機冒險者。有時你可以想到幾個同樣合理的可能性，需要一些幫助做出公正的決定。你可以將牌組作為隨機神諭，幫助你做出選擇。向你的桌友展示這種隨機性，這會讓你看起來更「公正」。藉由將冒險者的命運交給牌組決定，你可以緩解一些最強烈的不公平感。

以下是在遊玩流程中處理隨機事件的幾點提示。

:::note[你手上拿著什麼？]
有一個問題能讓你更好地裁決所有其他玩家行動：「你現在手上拿著什麼？」這是你 GM 工具箱中最重要的問題。冒險者嘗試的任何事情，都可能因這個答案而受阻或得到幫助。

**範例一**

玩家：我可以爬上這裡的牆壁，從上面往下看嗎？

GM：嗯，牆壁是用碎磚砌成的，所以應該很容易。你身上帶著什麼？

玩家：我把劍和盾牌放在地上。

GM：好的。那麼，你把劍和盾牌靠著牆放好，然後開始爬牆？

**範例二**

玩家：好的，我想用開鎖工具試試那扇有臉的奇怪門。

GM：誰持有光源？（這是「你手上拿著什麼？」的變形問法）

玩家 2：是我。我站在第一個房間守衛，確保沒有人偷襲我們。

GM：如果你要撬鎖，但光源在洞穴的入口，你基本上看不到自己在做什麼。肯定有劣勢。

玩家 3：我把我的火炬從玩家 2 那裡點燃，然後站在鎖著的門附近提供光線。

GM：好的，可以。以錢幣進行命運考驗。

冒險者手上拿著什麼的虛構情境，為接下來的整個場景提供了即時背景。
:::

### 以冒險者為目標的事件

有時，絞肉機事件針對某位特定的冒險者。這在情境下應該合理：行軍隊伍中的第一個人可能意外觸發陷阱，而隊伍末尾的人可能沒有注意到自己的藥水瓶破裂並在漏液。

有時，絞肉機事件或敵人的攻擊針對隨機冒險者。在此情況下，查看小阿爾卡那棄牌堆的頂端牌。隨機效果將針對其道途符合該花色的玩家。若桌邊有多位相同道途的玩家，利用牌面數值以快速方式在他們之間選擇（例如：奇數為你左邊的第一位玩家，偶數為你右邊的第一位玩家）。

### 以裝備為目標的事件

你會注意到冒險者卡有四個腰帶格位和二十一個背包格位。這些數字是刻意選擇的。

若你需要用絞肉機事件針對玩家的腰帶格位，瞥一眼小阿爾卡那的棄牌堆，注意頂端牌的花色，對應隨機數字 1 至 4：

- 寶劍：第一腰帶格位
- 錢幣：第二腰帶格位
- 聖杯：第三腰帶格位
- 權杖：第四腰帶格位

若你需要針對冒險者的背包，改為瞥一眼大阿爾卡那的棄牌堆，根據牌面數值對應 1 至 21 的數字。

## 寶藏有多少？

你也許會問：「我應該給玩家多少寶藏？」這樣的思維方式並不正確。財富與生命都廉價，且可以迅速失去。請記住，城市中的奢華生活費是 100 金幣（[城市階段](/city-phase/)）。達到這個目標可以感覺是一項重大獎勵。若玩家大豐收，讓他們享受幾個城市階段的奢靡生活，能強化他們完成重大成就的感受。錢花光了，他們就會回歸更為樸素的生活。

## 你手上拿著什麼？

### 可以看到什麼？

當玩家詢問他們的冒險者能看到什麼時，描述房間中他們可能看到或注意到的一切。這為玩家提供足夠的背景，讓他們能真正以有意義的方式與房間互動。

這些資訊當然受到幾個因素的過濾。冒險者也許無法在黑暗中看清，但仍能聆聽環境的聲音。他們無法看到隱藏或遮蔽的物體（例如秘密門或躲在窗簾後的竊賊），但可以自由地以更深入的方式搜索。

本質上，根據冒險者觀察周遭的仔細程度，有三個資訊層次。無需考驗，也不需要抽牌；唯一被考驗的，是玩家在提問時的思考深度。

1. **掃視**：涵蓋任何擁有充足光源的人在該區域中能感知到的一切。
2. **審視**：需要玩家提問才開放。「石像鬼是石頭製的嗎？」「書是什麼顏色？」「書頁是用什麼材料製成的？」「鉸鏈在內側還是外側？」
3. **調查**：需要冒險者與環境互動才開放。「我試著扭動那個麋鹿頭。感覺有沒有鬆動的跡象？」「我掀開掛毯。後面有什麼東西嗎？」「我敲了敲牆壁。聽起來是空心的嗎？」

資訊極少應對玩家隱瞞。慷慨地提供資訊：玩家得到的資訊越多，他們能做出的選擇就越有趣。因此，即使是在有秘密門的房間裡，冒險者也可能感受到房間裡輕微的氣流，暗示有些事情不對勁。或者如果窗簾後面有竊賊，他們可能聽到一聲壓抑的咳嗽，之後便歸於寂靜。

### 知道什麼？

當玩家詢問：「我的角色會知道 X 嗎？」時，快速判斷這個問題的冷僻程度。若問題是他們的冒險者可能知道的內容，慷慨地談論這個主題。若問題相當冷僻，詢問玩家是否有可以用來投注學識的主題（[冒險者進階](/adventurer/advancement/)）。

本質上，知識有兩個層次：

1. **顯而易見的知識**：涵蓋所有人都知道的事物，以及確立一個情境的一般預期。給玩家大量資訊，確保玩家和 GM 站在同一起跑線上，讓玩家做出明智的選擇。
2. **冷僻的知識**：某些專業領域只有特定人士才知曉。冷僻知識可能包括超自然生物的習性、組織內特定人物的喜好與厭惡，或諸界現實的形而上學。此類資訊只應提供給投注學識的玩家。

若玩家投注學識且你接受問題，就你所知盡量提供有關該主題的資訊——持續談論即興發揮，直到你說無可說。

## 主持挑戰階段

挑戰規則以公平且有趣的方式，解決敵對勢力之間的時間順序問題。「我們全部跑過那些小惡魔並撞上門」和「我在巫妖完成儀式之前向他射箭」等行動，最好透過挑戰階段的時間順序來解答。

探索階段可以自然地過渡到挑戰階段，也可以反向過渡。當局勢緊張，且敵對情況似乎無法通過單次命運考驗來解決時，進入挑戰階段。當對手被擊敗、挑戰結果已明確，或玩家希望撤退時，過渡回探索階段。

### 創造有趣的地點

可能的話，挑戰應在某個很酷的地方發生。

酷炫的遭遇空間蘊含潛在能量——它為玩家（以及 GM！）提供了互動的素材。在每個區域散布一兩個可互動的場景元素，包括動態地形、感官資訊和可使用的物件。

**地形**
創造能講述故事的空間。挑戰往往佔據一個臨界空間，介於安全與危險、穩固與不穩、固體與液體、冷與熱、高與低之間。善用這些對比。

**感官資訊**
冒險者對環境的理解，受限於其感官的極限。預設情況下，地下世界是一個黑暗且充滿困難的地方。對公會的視覺、聽覺和其他感官施加進一步限制，提供額外的挑戰。

**物件**
空房間令人乏味。每個空間都應散布有趣的物品。場景道具可以被撿起、躲在後面，或砸穿。

| 地形 | 感官資訊 | 物件 |
|------|---------|------|
| 橫跨深坑的橋梁 | 瀰漫於房頂的濃煙，使高大角色窒息或失明 | 垂掛的掛毯 |
| 45 度角倒塌的方尖碑 | 受碰撞時會噴出令人窒息的孢子、光或聲音的真菌 | 火炭盆 |
| 從天花板垂下的鎖鏈 | 貼近地板的霧氣，遮蔽矮小角色 | 絲質屏風 |
| 冰滑的地板 | 震耳欲聾的巨大鳴鐘 | 桌椅 |
| 可攀爬的神像 | 從明亮紅寶石發出的刺眼強光 | 書架 |
| 垂吊的吊燈 | 半充滿混濁水的區域 | 標本怪物 |
| 顫動搖晃的地板 | 幻覺物件與生物 | 噴泉 |
| 油污地面 | 尖嘯的風聲，阻礙飛行或投射武器 | 武器架 |

## 創造有趣的戰鬥

作為 GM，你的職責是讓挑戰順暢進行。你追蹤先攻順序，主導規則討論，並在玩家嘗試不同類型的行動時進行規則裁決。而且，重要的是，你在挑戰中控制著反派。

以下是試玩期間總結的幾點教訓，有助於你創造戰術性強且令人滿足的挑戰。

### 圍攻玩家

玩家手中的牌總量多於 GM。他們行動的頻率更高。然而，在反派的數量上，你具有優勢。

- 擁有多個敵人，你每次使用的行動可以針對同樣數量的冒險者。若五隻哥布林使用攻擊行動，這將影響多達五位不同的冒險者。
- 敵人也可以集中火力，攻擊同一位冒險者。善用[衝突行動](/challenge-phase/actions/)關於群體的規則。
- GM 的「陣營」通常能比玩家承受更多傷勢。

訣竅在於：玩家在橫掃殭屍、強盜或其他群體時，會感到很酷。同時，這也是一場消耗戰——GM 可以用大批敵人向玩家施壓。

### 盡量頻繁使用次要行動

在選牌方面，你比玩家擁有更大的自由度，因為你不必擔心牌的花色。因此，盡量頻繁地使用次要行動。

聰明的玩家會試圖用蠻力搏擊和各種效果將怪物「硬直鎖定」。聰明的 GM 則會使用恢復行動——它沒有要求的數值門檻，只需棄置任意一張輕微厄運牌作為次要行動，讓你的反派保持自由並隨時可以行動。

### 骯髒地打，公正地打

不要手下留情，但也要公正地打。不要在戰鬥進行到一半時更改怪物的數值。若玩家想出了一個精彩的計畫，並將你的大壞蛋推下塔樓，就讓它發生。當然，對鵝有利的事，對公也有利。若玩家開始將你的炎魔推下卡薩都-頓橋，你也可以同樣輕鬆地將他們的巫師推下去（如果你有合適的牌的話）。

若玩家正在落敗，他們必須做出選擇：堅持戰鬥、逃跑，或用嘴巴脫身。立場和士氣規則確保玩家在幾乎任何階段都有充分的機會用言語化解一場戰鬥。

## 創造有趣的反派

僅僅由「把對方打死」組成的挑戰可能令人乏味。設計以非擊敗敵人為目標的戰鬥。為此，創造擁有潛在能量的反派。

舉例來說，若一個絞肉機遭遇僅列出「1–4 具骷髏」，挑戰階段將只是敵對雙方互相打擊，直到一方因傷勢倒下。然而，若絞肉機詳細描述：「1–4 具骷髏在一個坑中，似乎在守衛一具戴著金冠的骷髏；戴冠的骷髏沒有移動」，遭遇就有了更豐富的質感，引發了幾個問題：

- 冒險者能安全地取下骷髏王的王冠嗎？
- 若他們這樣做，會觸發什麼嗎？
- 這些不死生物的故事是什麼？它與地下世界的這個區域有何關聯？
- 冒險者如何安全地穿越那個坑？

隨著公會穿越地下世界，一個反派的潛在能量將轉移到另一個身上，直到最終發生爆發——而玩家恰好夾在其中。

## 製造驚喜

若情況開始停滯不前，引入某些能大幅改變挑戰的事物。讓潛在能量轉化為動能。

以下是一張表格，為不同的震場方式提供靈感。瞥一眼小阿爾卡那棄牌堆頂端，以獲取隨機事件：

| 牌面 | 意外轉折 |
|------|---------|
| I | 當前衝突將附近一面牆砸倒，使當前房間與相鄰房間相連。那個房間裡住著什麼，它將如何與當前場景互動？ |
| II | 一隻遊蕩的怪物漫步進入場景。 |
| III | 一隻幼小的遊蕩怪物漫步進入場景，並將一位隨機冒險者認作親代。 |
| IV | 一支正在應付自身麻煩的競爭冒險者隊伍，意外衝入場景。 |
| V | 當前場景中的某物被揭示只是幻覺。 |
| VI | 場景中某物在進一步調查後，被揭示與原本所見截然不同。那個獸人不過是三個哥布林穿著一件長外套。石像鬼其實只是一尊雕像。 |
| VII | 一條秘密通道被揭示，通往隱藏的寶藏或危機。 |
| VIII | 一次偶然的打擊撕裂一個背包，導致兩瓶煉金藥水摔落在地。當兩種試劑混合，一隻可怕的黏液開始成形，在此過程中噼啪放出魔法雲霧。 |
| IX | 無形的靈魂鑽入附近陣亡敵人的身體，將其復活為肉偶。這些靈魂各有議程。它們想要什麼？ |
| X | 掉落的火炬或翻倒的火炭盆引發火災，濃煙和火焰迅速在區域內蔓延，成為一項危險。 |
| 侍從 | 區域突然被水、葡萄酒、污泥或其他液體淹沒。這場洪流的來源是什麼？ |
| 騎士 | 下一次有冒險者承受傷勢時，他們陷入昏迷。待他們清醒後，清晰地記得一個超自然的聲音曾對他們說話。這是什麼神啟？是真實的嗎？還是他們眼睛後方那個不可知的空間正在發生某些尚未診斷的異常？ |
| 王后 | 死神現身，靜靜地站在角落裡。他早到了幾分鐘，但願意等待。 |
| 國王 | 一群替身人形生物衝下通道，意外地撞入公會。他們纏成一團，互相絆倒。混亂結束後，隊伍的人數大幅增加。每位玩家至少有兩個、可能多達四個新冒險者。就連玩家也不確定誰是誰。給他們分配新的冒險者卡，直到情況釐清。 |
