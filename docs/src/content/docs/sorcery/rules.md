人類知道自己所感知的只是現實的一個狹窄切面。他們的狗能嗅聞到他們的鼻子所無法察覺的氣味。他們的貓能在他們的眼睛無法穿透的黑暗中視物。他們知道，對於某些事物，他們實際上是盲目的。這令人恐懼。

這些入侵從不溫和。正如水與油互斥，人類的界域與靈魂的界域彼此相剋。當它們交互之時，宇宙本身因這種錯誤性而扭曲、撕裂。這同樣令人恐懼。

就在人類感知的邊界之外，存在著不同的世界、不同的現實。有時——僅僅是有時——來自這些界域的力量會越過門檻，真正進入我們的世界。

然而，某些人卻甘願參與這場宇宙褻瀆。他們試圖看穿這個世界的面紗。他們向那些潛伏在門檻之上的事物打開大門，並邀請它們進入。

這些人是術士。沒有人喜歡他們。

## 遠界

有四個靈魂界域，有時被稱為「遠界」，與人類、矮人、狗、牛以及所有其他凡間生物居住的肉身界相切。這四個界域分別是荒域、林野界、詭異界與穹蒼界。所有魔法都源自這些界域之一。

### 荒域

荒域是一片荒蕪之地。它死寂。它即是死亡。那裡灰天籠罩、瘴氣瀰漫的沼澤延伸至地平線，散布著比凡間世界創生更早的古老戰爭留下的骸骨堆積。

- 暗光
- 恐懼
- 塑肉術
- 詛咒術
- 死靈術
- 喚屍
- 臭雲
- 枯萎

### 詭異界

詭異界是一個不斷變化的地方。它即是夢境。它即是瘋狂。詭異界的領域半是寓言、半是謬誤。夢者有時在睡眠中造訪詭異界；先知半活在詭異界、半活在肉身界。

- 幻象
- 鏡中融合
- 口袋洞穴
- 遙視
- 遮蔽
- 沉睡

### 林野界

林野界是一片狂野之地。它生機盎然。它即是混沌。那裡扭曲的藤蔓纏繞著深入不可思議之深的根系，蔓延的枝條伸入無形而翻騰的雷雲之中。

### 穹蒼界

穹蒼界是一個明亮而可畏的地方。它即是秩序。它即是法則。那裡藍天高聳，彩虹之橋通往雲端城市，有翼女子將蜂蜜酒奉給祖先之王，後者無止盡地征戰，以備末日之戰。

## 施法

當來自遠界的能量進入肉身界時，它灼燒我們的現實，引發奇異、不可預測且激烈的反應。這被稱為法術。

術士與法術打交道。他們研習法術、學習法術、交易法術，並加以運用。若你擁有權杖道途的「……之魔法」天賦，便可施放法術。

法術藉由使用稱為材料的象徵性與符號性物件，被召喚進入我們的世界。

- 材料在使用後不會被消耗。若材料是粉末之類的物質，術士只需使用少許，便足夠無數次施法。
- 材料可能被損毀、遺失或被盜。

術士使用「準備材料」城市行動（[城市行動](/city-phase/city-actions/)）以裝滿背包材料。術士準備材料時，可以依意願將盡量多的法術材料裝入背包，每件材料佔用一個背包格位。

施放法術時，術士手持材料，並花費 1 點或以上的決心。

- 你可以對任何你能觸碰或看見的目標施放法術。
- 你必須能夠大聲清晰地說話。施法非常顯眼。
- 若所使用的天賦處於訓練中，你還必須花費 1 點 XP 才能施放法術。

鐵能抵抗魔法。這是因為鐵是肉身界的骨骼。

- 施放在鐵製物品上的法術會失敗。
- 施放在攜帶大量鐵的目標上的法術也會失敗。
- 若你攜帶大量鐵或鋼鐵（鋼是鐵合金），你無法施放法術。

鐵對法術的影響在[法術規則](/sorcery/rules/)有進一步說明。

總結而言，施放法術你必須：

- 花費 1 點以上的決心；
- 以一隻手握持法術材料；
- 身上不得攜帶或穿戴大量金屬；
- 能夠大聲說話；以及
- 能夠看見或觸碰你的目標。

### 在探索中施法

在探索階段，你可以輕鬆施放以盟友、無人看守的物品或環境為目標的法術。這無需進行考驗，只需花費決心即可。

若你的法術以不願意的生物或其所攜帶或穿戴的物品為目標，你必須花費 1 點決心並進行權杖考驗。若成功，法術生效。若失敗，目標迴避或抵抗了法術。

### 在挑戰中施法

在挑戰中，你必須花費 1 點決心並吟誦咒文才能施放法術。以環境為目標施法時，行動數值無關緊要——法術會立即顯現。以不願意的生物為目標施法時，吟誦咒文的數值必須超過目標的先攻。

:::note[在地下世界採集材料]
資源豐富的術士或許可以（雖然罕見且困難）在地下世界中準備材料。這需要他們找到適當的所需物品，並在遊戲進行過程中以儀式方式準備它們——大概是作為一個紮營行動。GM 視情況進行裁定。
:::

## 魔法效果

每個分支的法術共有十種。

法術描述通常較為簡略。它們設計得多功能，以讓術士能找到聰明的使用方式。

- 也許「疾風」能把關閉的門炸開。
- 也許「閃光彈」能暫時奪去美杜莎的石化凝視。
- 也許你可以透過對自己施放「縮放術」逃脫枷鎖。

某些法術根據施放時花費多少決心而具有不同效果。例如，「臭雲」法術每花費 1 點決心便填滿 1 個區域。

- 術士在施放法術時選擇想花費多少決心。
- 若法術指定單一目標，術士每多花費 1 點決心，便可增加 1 個額外目標。
- 有時可以與 GM 商量，通過額外花費決心來換取額外效果或更大範圍目標。

這些用途並未被明確列出，但看起來是合理的法術應用。GM 被鼓勵與玩家共同界定法術的參數。

若干法術涉及區域（[衝突階段](/challenge-phase/)）的概念。區域是在挑戰和探索中劃定不同概念區域的便利方式。GM 應在這些法術的區域概念尚未建立時，說明其運作方式。

### 專注

某些法術即時生效後便結束。某些法術只要術士保持專注——持續喃喃低語咒文並集中意志——便會持續產生效果。你每次只能專注於一個法術。

若你在專注法術時受傷，除非通過權杖考驗，否則你將失去法術並引發禍兆。在挑戰中，這需要使用命運考驗行動。

### 禍兆

魔法出錯時，抽取相應的禍兆表（第 202–203 頁）。這代表遠界能量對肉身界的災難性入侵。

以下情況會導致魔法出錯：

- 術士在專注法術時受傷
- 兩個法術相互抵消
  - 若你成功，你維持住法術。
  - 若你失敗，你失去法術並引發禍兆。
- 術士在施法過程中被打斷
- GM 裁定的其他情形

### 鐵：反魔法

鐵是肉身界的核心。因此，它與靈魂和魔法都是對立的。

魔法無法影響鐵製物品。任何法術都無法施放在鐵塊、鐵像、鐵劍或鐵盾上——法術會如同被反制一般消散。

若你身上任何地方攜帶了相當體積的鐵或鋼鐵，你便無法施放法術。

什麼算是相當體積的鐵？一枚硬幣是可以的。一口鑄鐵鍋不行。一把鋼劍不行。金屬護甲絕對不行。

若某人穿著覆蓋身體至少 75% 的鐵製護甲，他們便無法成為任何法術的目標。（魔法仍可能對他們產生影響——由法術召喚的火牆依然會燒灼全副鎧甲的騎士，但騎士本人無法成為天火法術的目標。）

當禍兆發生時，情況就變得有趣了。

:::caution[身體恐怖與令人不安的內容]
許多禍兆效果會改變或改造冒險者的身體，有時方式令人不安。某些玩家會覺得這很有趣；另一些則未必。

處理敏感議題並與玩家協商舒適邊界，是 GM 需要的社交技巧之一。當有人提出顧慮時，請真誠地與他們一同設法解決。在第 0 場次時便坦誠說明這種可能性，並願意調整禍兆在你們桌邊的運作方式，以照顧同桌玩家的感受。善待彼此。
:::

## 林野界禍兆

| 結果 | 效果 |
|------|------|
| I | 一隻禿頭猿爬入現實。牠自行剖腹，將腸子討好地呈獻給術士，帶著始終不散的微笑死去。 |
| II | 尖嘯菇在術士四周迅速盛開。任何人在此區域中移動都會引發巨大喧譁，在絞肉機表上觸發一次隨機遭遇抽牌。 |

| V | 術士所攜帶的所有藥水爆裂。瓶子損毀。冒險者記錄表上與藥水相鄰欄位的物品遭到毀壞。 |
| VI | 閃電從術士體內迸發，對房間內所有攜帶金屬者（不論友敵）造成傷勢。 |
| VII | 此地區絞肉機表上的所有動物都被吸引至術士身邊。下一個時段自動發生一次與所有周遭動物的遭遇。 |
| VIII | 術士附近的所有木製物品（魔杖、門扉、箱子、法杖、火炬、弓等）扭曲變形，宛如被巨人之手折彎。此室中所有木製物品均損毀。木門或箱子上的陷阱被觸發，影響距其最近的人事物。 |
| IX | 術士的影子離開，去從事某種惡作劇。待牠惹出某種麻煩後才會回來。在此之前，術士沒有影子。 |
| X | 術士立即開始嘔出一個寄生雙胞胎的碎塊：小手、眼睛、毛髮、牙齒。在整個過程結束之前，術士幾乎無法行動。在挑戰中，這使術士陷入暈眩效果。此禍兆持續整個時段。 |
| 侍者 | 術士的腿部向後扭轉並長出毛皮，形似山羊腿。此變化為永久性。 |
| 騎士 | 術士皮膚上長出一種厚實肉質的真菌，如同無恥的癌腫蔓延。此效果為永久性，直至疾病被治癒為止。食用此真菌會令人上癮。 |
| 王后 | 術士受到狼人詛咒。當月光照耀時，他們發生變化：過程混亂、失控且暴烈。除非找到解藥，否則狼人詛咒是永久的。 |
| 國王 | 公會成為狂野獵夜的目標。一位地精禮貌地通知公會，他們有一天的時間逃跑，之後一支由木精靈組成的軍隊將竭盡所能追殺他們。祝好運。 |

## 詭異界禍兆

| 結果 | 效果 |
|------|------|
| I | 術士失去聲音。在愚者牌令牌組重新洗牌之前，術士承受噤聲狀態。此後，術士的親屬發誓曾在黑暗中聽見術士在呼喚他們。 |
| II | 由無形之手用術士周圍任何可用材料迅速拼成的榮格原型夢魘偶像，在四周現形。 |
| III | 房間內所有人（包括術士）必須立即進行權杖考驗。任何失敗者均承受昏迷效果，陷入沉睡。 |
| IV | 術士所在區域內所有人（包括術士）立即以隨機立場承受激勵效果（[社交遭遇](/crawl-phase/social-and-example/)）。依情境進行角色扮演。 |
| V | 術士後腦勺處裂開一個嘴巴，用泰爾威斯語喋喋不休。GM 立即向所有能聽懂的人傳遞一則謠言。這個流著口水的嘴巴會一直存在，直到愚者牌令牌組重新洗牌為止。 |
| VI | 術士開始流下黏稠帶鹹味的眼淚。他們也從眼中流出幾隻小型黏液獸，爬走後吃東西並逐漸長大。 |
| VII | 術士的皮膚、器官和肌肉變得隱形。在愚者牌洗牌之前，術士看起來像一具活動的骷髏，錯綜複雜的神經和血管暴露在外。詭異極了。GM 扮演的角色對此感到毛骨悚然。 |
| VIII | 術士成為一個活幻象。幻象不過是全息投影。他們無法與肉身界互動或施放法術，但可以看見並被看見。然而，一切幻象對術士而言都是真實的。一把幻象劍可以切傷他們，一條幻象龍可以吃掉他們。此效果持續至愚者牌洗牌為止。 |
| IX | 術士只能以押韻方式說話。他們在角色扮演中說的每一句話都必須以押韻對句的形式表達。此效果持續至下一個紮營階段結束後。 |
| X | 術士最糟糕的噩夢在下一個紮營階段步驟 3 時，於肉身界中實體顯現。GM 和術士的玩家應事先就此內容進行討論。 |
| 侍者 | 術士的膚色褪去。其頭髮、皮膚和虹膜變成乳白色。此變化為永久性。 |
| 騎士 | 術士在額頭正中央長出第三隻眼。每當他們使用幽法時，這隻眼睛便會睜開。否則不易察覺。此變化為永久性。 |
| 王后 | 一隻食魔靈——一種遮蔽且無形的吞噬魔法之靈——開始跟隨術士。術士必須設法驅逐或擊敗它，否則任何非專注法術都會提前結束。 |
| 國王 | 術士被永久詛咒為噤聲。然而，當他們說話時，幻象文字會在其頭頂短暫漂浮，形如對話氣泡。所幸，他們仍可施放法術。 |

## 穹蒼界禍兆

| 結果 | 效果 |
|------|------|
| I | 坐在術士左右兩側的玩家，其冒險者的所有狀態均被治癒。術士承受同等數量的傷勢。（若此舉殺死術士，他們將被封聖。） |
| II | 術士皮膚上浮現痛苦的膿瘡，這是未稀釋天界輻射的副作用。術士承受一次傷勢並承受壓力狀態。 |
| III | 此室中的口糧變得奇異地污染：水變成血液，乾肉變成跳動的血肉。只有純素口糧不受影響。 |
| IV | 長著堅硬金屬外殼與人臉的神聖蝗蟲成群湧現。牠們摧毀整個地城層的所有植物與植被。 |
| V | 術士周圍顯現天使讚歌。音樂優美卻嘈雜。潛行變得不可能。讚歌在愚者牌下次令牌組重新洗牌後平息。 |
| VI | 術士顯現彌特里斯的悲憫聖傷。他們承受疲憊狀態，直到使用城市行動前往彌特里斯聖殿祈禱為止。 |
| VII | 術士所在房間內所有武器（明確以鐵製成者除外）永久轉變為大小或形狀相近的工具。例如，一把劍可能變成犁鏵，一支矛可能變成修剪鉤。 |
| VIII | 術士眉心燃起光環。所有人都印象深刻，唯有靈魂除外。所有靈魂對術士產生自動的負面反應，在戰鬥中，他們會優先攻擊術士。此光環持續至術士使用城市行動前往彌特里斯聖殿祈禱為止。 |
| IX | 附近顯現一隻幽靈之手，開始在附近表面以維圖斯語鐫刻預言，預告除非滿足一系列非常具體（且不太可能實現）的條件，否則厄運將至。 |
| X | 公會成員返回城市後，凡為父母者均被告知其長子/長女已亡。據報，術士的名字以盧恩文字刻寫在亡子的額頭上。 |
| 侍者 | 術士身上的貴重硬幣與寶石永久變成聖甲蟲。貪婪是可憎的罪惡。 |
| 騎士 | 一座活動雕像從石頭中爬出。它與術士幾乎一模一樣，是石質的孿生體。它的使命是殺死並取代術士。 |
| 王后 | 術士懷胎八月，無論他們是否具備正常所需的生理條件。懷孕期間，他們承受疲憊狀態。若妊娠足月，一個半神孩子將降臨世界。 |
| 國王 | 天空中出現一顆棕色汙穢之星。第三夜結束時，一顆巨大的隕石摧毀城市的一個隨機街區，使城市在數週內無法居住。 |
