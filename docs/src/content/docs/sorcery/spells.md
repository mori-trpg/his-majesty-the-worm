## 荒域法術

### 腦熱

**材料：** 由野馬鬱蘭、百里香、馬鞭草與香桃木葉製成的粉末香囊

術士取一撮材料粉末，向法術目標吹去。目標進入暴怒狀態。

- 目標在攻擊時獲得優勢，但必須始終以手中抽到的最低點數牌參與先攻。
- 不具情緒感受的生物（如無意識的造物）不受此法術影響。

此法術需保持專注。

### 暗光

**材料：** 一名被吊死的殺人犯的醃漬左手

醃漬之手握著一根蠟燭，在此法術生效期間不會熄滅。這根蠟燭只為持燭者提供光明，他人無法看見，使目標得以在保有完美視野的同時悄然移動。

- 此蠟燭忽略磨刀霍霍表上的「火把熄滅」結果。
- 每額外花費一點決心，可讓另一人也能看見暗光。

此法術需保持專注。

### 操控不死

**材料：** 以枝條編織、以新月之夜自墓地採摘的草綁定的小人偶

此法術讓術士對一個不死生物施加控制效果，傳達一道命令。

- 此命令的字數上限等於術士的權杖屬性值。
- 每額外花費一點決心，可將命令延長等同於術士權杖屬性值的字數。

此法術持續至命令執行完畢。用於施放此法術的決心在法術持續期間無法補充。

### 恐懼

**材料：** 以被謀殺的女性的頭髮為燈芯、以黑色蛇脂製成的蠟燭

術士編織一個只有目標才能看見的幻象，使其將第二個自願的目標視為令人毛骨悚然的怪物。主要目標受到激勵，對幻象覆蓋的生物懷有恐懼之情。

- 恐懼的生物將試圖逃離恐懼的來源，除非逃跑確實需要，否則避免戰鬥。他們在先攻時會優先打出高點數的牌。
- 不具情緒感受或不受幻象影響的生物，不受此法術影響。

此法術需保持專注，但任何改變目標立場的行動也會終結此法術。

### 塑肉術

**材料：** 一把銀色阿塔姆匕首

術士用銀刃切下自己身體的一個部位，被切下的部位能夠自行移動，但動作笨拙。

- 若被分離的身體部位遭受傷害，目標承受一點穿刺傷勢，法術隨即終結。
- 若被分離的部位放回原來的位置，它將自願與目標的身體重新結合。
- 被分離的手掌可像蜘蛛般爬行。它非常難以被發現。它可以相當輕易地掐死熟睡之人，或將毒藥滴入杯中。
- 被分離的眼睛可以在地上滾動。目標閉上眼睛即可透過被分離的眼睛完美視物。
- 被分離的耳朵在地上如魚般撲騰。目標可透過它完美聆聽。
- 被分離的嘴巴可正常說話。

此法術需保持專注以維持被分離身體部位的活動。用於施放此法術的決心在法術持續期間永遠無法補充。

### 詛咒術

**材料：** 一個埋在貧民墓中的吝嗇鬼的醃漬眼睛

目標承受一個隨機詛咒——在下面的表格上抽牌以得知是哪種詛咒。術士通常用此法術迫使他人完成任務或服從其意志。

與其他法術不同，詛咒持續生效，直到術士解除（可隨時為之）或被反制法術破除。詛咒無法透過恢復行動解除。

此法術無需保持專注。用於施放此法術的決心在法術持續期間永遠無法補充。

| 牌面 | 詛咒效果 |
| ---- | -------- |
| I | 狗對你充滿敵意。你所遇到的任何狗都會自動嘗試攻擊你，並將你列為首要目標。城市階段結束時，你將承受踉蹌狀態（因被撕咬），或壓力狀態（因你整段時間都在躲避狗）——各佔五成機率。 |
| II | 你飽受噩夢侵擾。每次入睡，你有50%的機率因夢魘而醒來並承受壓力狀態。 |
| III | 你所吃的食物有50%的機率在入口時變成灰燼。若發生此情況，你將浪費那份口糧。 |
| IV | 你厄運纏身。你承受疲憊狀態。 |
| V | 你呈現出一具乾癟屍體的外貌。大多數GM角色都會對你抱持敵意。沒有人想與你共寢。但好的一面是，大多數不死生物都會無視你。 |
| VI | 你被詛咒只能以押韻方式說話。你在角色扮演中說的每一句話都必須押韻。 |
| VII | 你的武器容易生鏽損壞。每當你出一張點數10或以上的牌發動攻擊，你的武器承受一個缺口。木製武器可承受1個缺口，鐵製武器可承受2個，鋼製武器可承受3個，超過即損壞。 |
| VIII | 你的性別特徵發生了改變。很奇妙（但也許令人興奮？）。 |
| IX | 老鼠與各種害蟲愛上了你。牠們到處跟隨你。牠們想睡進你的嘴裡。每天晚上，有50%的機率牠們會從你的背包中拿走一件你不需要的東西，換成垃圾。不過是還不錯的垃圾。 |
| X | 你每踏出一步，都伴隨著一聲虛無縹緲、無形的鐘聲，同時有個聲音說「羞恥」。潛行成為不可能。 |
| 侍者 | 你病弱纏身。清除帶傷天賦與受傷狀態時，你必須燃燒兩個充能羈絆，而非一個。你（遺憾地）對治療效果免疫。 |
| 騎士 | 你在詛咒期間衰老——如同枯槁的老太婆。你醜陋而無吸引力。白內障使你無法看清三十英尺以外的事物，你也無法使用衝刺行動。 |
| 王后 | 你只能說泰爾威斯語。那是你懂的語言嗎？不是？那你聽起來就像在說胡話。 |
| 國王 | 當你進入緊張情況（如戰鬥）時，進行權杖考驗。若失敗，你變成一隻雞，持續一個回合（或相當時間）。 |

### 死靈術

**材料：** 一條刻有語言符印的醃漬舌頭

將材料舌頭置入一個死者的頭骨中。術士現在可以與目標交談，彷彿它仍然活著。

- 目標並無義務主動助人或說出實情，但可以像正常人一樣進行交涉。死者有時在意完成未了的心願、獲得復仇、向摯愛道別，或其他種種事宜。

此法術需保持專注。

### 喚屍

**材料：** 秘銀墨水

術士花費一個時段，用秘銀墨水在屍體周圍描繪奧術符號。藉由這個儀式，術士召喚一個來自荒域的魔鬼爬入屍體，並將魔鬼束縛於其服務之下。這創造出一具殭屍（見附錄C，[怪物圖鑑](/dungeon-denizens/bestiary/)）。

- 殭屍沒有屬性值。它們只是腐爛的血肉。然而，它們通常具備生前所擁有的一切身體能力。
- 術士可以在挑戰中使用命令行動向已束縛的殭屍下達命令。殭屍將以真誠的善意和合理的智識服從大多數命令。殭屍甚至服從自殺式命令。
- 殭屍保持被束縛狀態，直到它服從了等同於術士權杖屬性值數量的命令或完成相應任務。
- 每額外花費一點決心，可獲得等同於術士權杖屬性值的額外命令數量。
- 當所需服務完成後，魔鬼將為了自己的目的佔有這具身體。殭屍屆時可能發動攻擊、蹣跚離去，或化為一堆灰燼。

此法術持續至殭屍完成其服務。用於施放此法術的決心在法術持續期間永遠無法補充。

### 臭雲

**材料：** 一個塞滿墓土的木乃伊化綠蛙

術士的口中噴湧出宛如千具開棺屍體的腐臭。術士所在區域或相鄰區域被臭雲籠罩。

- 在回合開始時身處雲中的所有生物均承受暈眩效果，在回合開始時少抽一張牌。
- 此效果可疊加。例如，若某人在雲中停留至第二回合，他們將再次承受暈眩，在該回合少抽兩張牌。若某人在雲中停留四個回合，他們將陷入昏迷。
- 術士可透過為此法術花費更多決心來擴大臭雲範圍。每額外花費一點決心，可多覆蓋一個區域。
- 無需呼吸的生物對此法術免疫。

此法術需保持專注。

### 枯萎

**材料：** 一罐各刻有衰敗盧恩符文的數百枚舊指甲

若施放於活著的生物身上，目標承受一點傷勢，如同被嚴寒緊緊攫住。

- 目標顯現出明顯的衰老跡象——灰白的頭髮、皺紋密布的肌膚、發黃的眼白、老人斑——直至傷勢癒合。

若施放於不死生物身上，目標不受傷害，但變得更加可怖。

- 殭屍化為骷髏。
- 骷髏化為幽靈。

若施放於物品身上，物品呈現出明顯的腐敗跡象。

- 食物腐爛；鐵鏈生鏽；木材發霉；石材不受影響。
- 物品承受一個缺口。若為此法術花費兩點決心，物品直接被破壞。

## 林野界法術

### 操控動物

**材料：** 一根以支配盧恩符文刻寫的獅骨製成的權杖

此法術讓術士對一隻田野、森林或泉澤中的動物施加控制效果，傳達一道命令。

- 此命令的字數上限等於術士的權杖屬性值。
- 每額外花費一點決心，可將命令延長等同於術士權杖屬性值的字數。
- 動物不會服從明顯的自殺式命令。

此法術持續至命令執行完畢。用於施放此法術的決心在法術持續期間無法補充。

### 閃光彈

**材料：** 巨型蝙蝠的鳥糞

此法術使一個光源劇烈閃爍後熄滅。蠟燭和燈籠事後可重新點燃；火把則不能。

- 蠟燭只是蹦出蠟油後熄滅；事後可再次點燃。
- 火把和燈籠會劇烈閃爍，使持有者承受傷勢。目標隨即著火。
- 篝火會引發如炸彈般的爆炸。將術士的吟誦咒文數值與區域內每個角色的先攻值比較。若術士的總值較高，生物承受失明效果，持續至目標下一回合結束，或相當時間為止。

### 深淵抗力

**材料：** 一個鈣化的美人魚胎兒

若施放於人形大小的生物，他們可以在水面上行走。

- 若施放於箱子大小或更小的物品，該物品可漂浮在水面上，即使沉入水中也不例外。
- 可以花費+1決心讓小型沉船（如划艇）浮起，花費+3決心讓大型沉船浮起。

此法術需保持專注。

### 疾風

**材料：** 刻有盧恩的甲蟲殼

伴隨著一陣狂風，將人形大小的目標位移至相鄰區域。

- 目標輕輕落地，移動不造成傷害，但可能被放置於危險地點（如酸液池中）。
- 每額外花費一點決心，可多增加一個目標。
- 術士可對自身使用此法術，在戰場上機動穿梭或跨越縫隙。

### 雷鳴

**材料：** 一片碎裂的雷霆鐵

術士吟誦的最後一個音節在術士能看見的一個區域中迴響如雷鳴。

- 這立刻震碎區域內所有易碎物品：未經淬鍊的玻璃、瓷器之類。
- 區域內所有生物（術士除外）必須鬆開手中持有的所有物品以雙手掩耳，否則將承受暈眩與失聰效果。

### 圖騰

**材料：** 從一棵古老橡樹落下的最後一顆橡果

術士將橡果放入自己（或目標）的口中。他們化身為一種獨特的動物形態，代表著其靈魂的圖騰。

- 處於此形態時，目標所有屬性值均為+0。

- 變身近乎瞬間完成，但不包含所穿衣物和攜帶物品，這些都會掉落在目標周圍。

將橡果吐出即終結法術。因此，每當目標想要說話、進食、飲水或用嘴取物，法術便告終結。

### 元素防護

**材料：** 以未山羊（Ungoat）的胃製成的皮革袋，裝滿珠子

法術目標對所選一種元素造成的傷害幾乎免疫。

- 火焰：目標及其裝備不受熱源或火焰造成的傷害，高達熔爐火焰的強度均無效。
- 水流：目標及其裝備不受寒冷或冰霜衝擊造成的傷害。
- 氣流：目標無需呼吸。
- 大地：目標及其裝備不受墜落造成的傷害。
- 每額外花費一點決心，可額外防護一種元素。
- 每額外花費一點決心，可多增加一個目標。

此法術需保持專注。

### 獸語術

**材料：** 一個木乃伊化的巨型蜘蛛蛋

術士將材料置於舌下。一顆巨型蜘蛛蛋約有雞蛋大小，因此會使他們的說話含糊不清，但動物能聽懂。術士現在可以像動物能夠正常說話一樣與之交談。

- 動物沒有特別義務協助術士，但動物通常都很樂於助人且誠實。
- 普通動物有些頭腦簡單。
- 話雖如此，動物非常在意自己熟悉領域內的事情，通常對那以外的事物毫不關心。牠們在意食物、交配與安全——幾乎什麼都不在乎。例如，綿羊很容易分辨自己羊群中的成員，但一隻綿羊幾乎不可能分辨兩個精靈的差別。

此法術需保持專注。

### 元素之牆

**材料：** 一個裝有蜜漬木乃伊的蝸牛殼

每為此法術花費一點決心，術士可以召喚一段10英尺×10英尺×2英尺，由四元素之一構成的牆。

- 土牆不透明且堅硬如石。
- 風牆不遮擋視線，但遠程武器或飛行生物無法穿越。
- 火牆不透明但可穿越——任何穿越的生物承受一點傷勢。
- 水牆不透明且不可穿越。只能在水體中召喚，但可阻止船隻和其他水上交通工具靠近。

此法術需保持專注。

### 木紋編織

**材料：** 一瓶林野人的血液

術士可以控制植物的生長和木材的形狀。

- 若施放於活著的植物，術士可以讓其迅速放大生長，或使其枯萎縮小回種子狀態。
- 若施放於木製物品，物品因木料扭曲變形而承受一個缺口。
- 若施放於有植被的場所，術士可以召喚纏繞的藤蔓，使一個區域內的所有生物陷入定根效果。
- 若提供等量大小的原材料，術士可以使用此法術瞬間塑造一件木製物品。

凡是可能出現在中世紀動物圖鑑中的，大概都算是動物。這包括海怪、梟熊和雞蛇怪，是的。但這不包括骷髏和傀儡。動物吃東西，也在林子裡方便。
:::

## 詭異界法術

### 賦靈物件

**材料：** 一瓶液態汞

術士命令一件物品執行其預定用途。匕首刺擊。鑰匙解鎖。鎬子挖掘。

- 此命令的字數上限等於術士的權杖屬性值。該物品服從命令，彷彿由一隻隱形的手操縱。
- 每額外花費一點決心，可將命令延長等同於術士權杖屬性值的字數。
- 無論命令中說了多少解釋，一件物品每次施法只能完成一項任務。鑰匙可以打開牢房的門，或你的手銬，或你朋友的手銬，但絕不能在同一次施法中完成三件事。同理，無人持握的武器每次施法只能攻擊一次。
- 在挑戰中，物品的行動數值等同於術士的吟誦咒文數值。

此法術持續至物品完成任務。用於施放此法術的決心在法術持續期間永遠無法補充。

### 縮放術

**材料：** 一條以鼠王尾巴編成的繩索

術士使目標放大或縮小。被改變大小的生物縮小或增大至原始身高的一半或兩倍。

- 當此效果有利時（擠入洞穴、潛行繞行），獲得優勢。
- 當此效果不利時（力量角力），承受劣勢。
- 大小差異永遠不影響攻擊數值，但可能影響蠻力拼搏等行動。

此法術需保持專注。

### 激怒

**材料：** 一條木乃伊化的毒蛇

術士編織一個只有目標才能看見的幻象，使其將第二個自願的目標視為模糊卻令人憎恨的敵人。主要目標受到激勵，對幻象覆蓋的生物懷有憤怒之情。

- 在戰鬥中，憤怒使人愚蠢而笨拙。憤怒的生物將竭力攻擊憤怒的來源，幾乎不顧周遭環境。
- 不具情緒感受或不受幻象影響的生物，不受此法術影響。

此法術需保持專注，但任何改變目標立場的行動也會終結此法術。

### 化虛為實

**材料：** 一面以白鹿皮製成、刻有盧恩的鼓

術士在施法時敲擊刻有盧恩的鼓。所在房間內所有無形或不可見的實體或物品，均變得有形且可見。

- 此法術的材料——一面鼓——為標準尺寸，但在挑戰中需雙手演奏。
- 每額外花費一點決心，可讓術士額外指定一個房間。

此法術在鼓聲持續時維持生效。

### 鏡中融合

**材料：** 一瓶從沉睡兒童眼中收集的淚水

術士將材料灑在鏡面上，在鏡中創造一個門戶，生物或物品可以穿越其中。

- 踏入鏡中的人變成倒影；他們只能在鏡中被看見。他們能看見並聽見鏡子附近所說的一切，但無法與外界互動。
- 鏡子必須足夠大才能讓人進入；一個半身人可以踏入相對較小的鏡子，而巨魔則需要一面真正大型的鏡子。多個生物可以作為倒影棲居於同一面鏡中。
- 棲居於鏡中的生物可以隨時踏回鏡門離開。
- 物品可以留在鏡中。它們在外界可見但無形，無法被觸碰。
- 若鏡子被打碎，棲居其中的冒險者被送回現實並承受兩點傷勢。鏡中的物品在回到現實時被破壞。
- 被「攜帶」於鏡中的物品因魔法重量仍佔用背包格位。

此法術持續至棲居其中的生物選擇離開或鏡子被打碎。

### 幻象

**材料：** 一個七面棱鏡

術士創造出任意物品或生物的幻象。

- 幻象不過是一個影像——全息投影——沒有重量或實質，不產生聲音、氣味等。
- 幻象只能增添，而非消除。例如，幻象無法讓某物消失，但一面幻象之牆可以遮擋它。讓某物消失是遮蔽法術的職能。
- 幻象服從術士的心念指揮。在挑戰中，這需要一個雜項行動。

此法術需保持專注。在維持法術期間，術士可以每花費一點決心為幻象增添一個細節。例如，術士可以花費一點決心，讓一個幻象騎士從其背包中取出一件新物品（如一卷羊皮紙或一把武器）。

### 口袋洞穴

**材料：** 一片直徑不超過九英寸的獸皮圓片，取自在午夜時分獻祭的黑色牛犢

術士念誦咒文並放置這片牛犢皮圓片時，它在任何無生命材料上形成一個開口。

- 法術生效期間，圓片本質上是一個洞——一扇窗——通往無生命材料的另一側。這在牆壁、門扉、寶箱上創造出一個可讓事物通過的開口。
- 此法術對活體組織完全無效。
- 介於有生命與無生命之間的模糊地帶（矽基病毒、不死生物和有生命的石質石像鬼）由GM自行裁決。
- 此法術永遠不會破壞被穿透表面的實際完整性。即使口袋洞穴被放置在關鍵承重位置，結構本身依然完好。
- 若無生命材料沒有另一側，會開出一個大約一英尺深的洞。放入此洞的物品在法術終結時會被周圍材料吞沒，但不受損傷。

此法術需保持專注。法術終結時，窗口縮回材料的牛犢皮圓片中。

### 遙視

**材料：** 一個水晶球

術士凝視水晶球，可以看見他們曾親眼造訪過的某個地點。

- 預設情況下，此法術只在所選地點位於術士目前所在形而上學區域內時有效——同一座城市、同一層地城、同一片森林等。
- 術士可以花費+1決心查看其當前形而上學區域之外的地點。

此法術需保持專注。

### 遮蔽

**材料：** 一個死於精靈之箭的人的喪葬裹屍布

術士將材料裹屍布覆在目標身上。目標及其攜帶的一切物品均進入遮蔽狀態。

- 遮蔽的生物在沒有魔法或特殊感知的輔助下，不可能被看見。通常，若遮蔽的生物靜止不動且保持安靜，無法看見他們的生物便無法刻意將其作為目標。
- 若遮蔽的生物移動，附近的生物可能隱約感知到其存在。所有針對他們的目標嘗試均承受劣勢。
- 遮蔽的生物對無法看見他們的生物執行傷害行動時，獲得優勢。

讓目標躲藏所需的幻象魔法，在目標與可見物品互動時會變得混亂且緊繃。若目標磨劍、從腰間取藥水飲下，或喬裝易容，隱形得以維持。若目標開門、向敵人射箭或替朋友綁鞋帶，則必須立即額外花費一點決心才能維持法術。

此法術需保持專注。

### 沉睡

**材料：** 由壓碎的蓮花籽、沙子與苦艾製成的粉末香囊

術士取一撮材料粉末，向目標吹去。目標受到一種超自然安眠劑的影響。

- 若目標不在緊張或危險的情況下，目標承受昏迷效果，只能以一記有力的巴掌喚醒。
- 若施放時花費4點決心，目標在睡眠中不會老化，也不需要食物。
- 若目標正處於緊張或危險情況下（如戰鬥中），目標變得昏昏欲睡並承受暈眩效果。

## 穹蒼界法術

### 預兆

**材料：** 一本《索菲亞法典》，神話教的聖書

術士藉由修煉扶乩——一種以書本為媒介的占卜形式——接受超自然的智慧，領悟預兆與徵兆的意義。這使他們能夠展望未來、防範厄運，從而迴避失敗或大失敗的後果。

若需要進行命運考驗，而術士有時間翻開《索菲亞法典》的隨機頁面施放此法術，GM將代替玩家為命運考驗抽一張牌。GM不向玩家展示此牌，而是以《索菲亞法典》中的寓言形式描述牌面內容。

- 例如，寶劍X可能被描述為：「書中如此記載：有一個人被刺中多次，面朝下躺著——也許傷勢很重，也許已然死去。他的臉背對著我們。」
- 術士的玩家利用其對塔羅牌圖象的非角色扮演知識，以角色身份詮釋這些結果。他們不得使用外部協助（不得查手機）。
- 法術施放後，公會成員可以選擇是否嘗試該任務。若決定嘗試，GM揭示牌面。
- 由於此法術在技術上發生於命運考驗之前，進行考驗嘗試的玩家可以選擇花費決心獲得優勢。他們也可以像往常一樣在此考驗中推動命運。
- 若公會決定不根據預兆採取行動，受縛於命運的規則仍然適用（[命運考驗規則](/basics/tests-and-fate/)）。牌被棄置。那個行動在情況發生重大變化之前，或成功或失敗。

此法術需保持專注。

### 束縛

**材料：** 一位埋在聖土中整整一年的聖者的指骨

術士施法時，手持材料作為避邪符，點名一個生物。術士視線範圍內所有被點名的生物均陷入定根效果。

- 若名字是具體的（「約翰·多伊」），法術影響所有同名者。
- 若名字是泛稱（「人類」），法術影響該類型的每一個生物。
- 依GM判斷，若施法指令模糊或令人困惑，可能引發禍兆。
- 此效果持續至術士的專注被打斷。受影響的生物無法使用恢復行動清除此效果。法術必須被反制或打斷。

此法術需保持專注。

### 魅惑

**材料：** 在象牙香爐中燃燒的沒藥油

術士編織一個只有目標才能看見的幻象，使其將第二個自願的目標視為某個不明確但值得信賴的朋友。主要目標受到激勵，對幻象覆蓋的生物懷有信任之情。

- 懷有信任立場的生物不傾向於戰鬥。若情況迫使戰鬥而未改變立場，生物可能試圖透過非致命手段了解公會的實力。
- 不具情緒感受或不受幻象影響的生物，不受此法術影響。

此法術需保持專注，但任何改變目標立場的行動也會終結此法術。

### 羽化

**材料：** 一根拉瑪蘇的羽毛

目標物品或生物漂浮起來，其重量化為羽毛般輕盈。

- 若正在墜落，目標輕輕飄落地面。術士可以快速施放此法術以回應墜落，但必須事先將材料握在手中。
- 若在地面移動，目標不受危險地形（如尖刺）造成的傷害，也不會觸動壓力板陷阱。
- 沉重的物品變得容易搬運。

此法術需保持專注。

### 守護圓環

**材料：** 裝在小銀聖體匣中的受崇敬聖人之灰燼

作為施法的準備，術士花費一個時段，用灰燼勾勒出一個半徑不超過十英尺的防護盧恩圓環。

當術士念誦此法術時，選擇一個遠界（荒域、林野界、詭異界或穹蒼界）。來自該遠界的生物與靈體無法穿越圓環或傷害圓環另一側的人。

- 若術士畫出圓環但未念誦咒文為其賦能，可以將其作為靈體的「陷阱」——等靈體無意中踏入盧恩圓環後再施放法術。
- 每額外花費一點決心，可再封禁來自另一個遠界的靈體。

此法術需保持專注。

### 守護天使

**材料：** 一瓶由處女從聖泉中汲取的清水

術士施放此法術時，召喚一個被稱為守護天使的隱形微型穹蒼界靈體。術士將此守護天使束縛於一個目標身上，保護他們免受單次攻擊。

- 在挑戰中，術士將用於吟誦咒文的那張牌交給目標。此牌面朝下放置在目標玩家面前。目標可自行選擇何時將牌面翻開，用其數值執行閃躲或反擊。
- 防禦行動的總數值可與同類型的其他行動累加。例如，若玩家已有一張面朝下的閃躲牌，而守護天使的牌也用於閃躲，兩次行動的數值將相加。
- 此法術不計入每位玩家面朝下一張牌的限制。但同一時間不能有多個此法術的效果疊加。

此法術持續至被使用。

### 天火

**材料：** 一根用獻祭公牛煉製油脂製成的供奉蠟燭

術士向目標噴射出一道天火。天火是一種明亮的火焰，只發光而不發熱。它不會燃燒所接觸的物品。

- 若施放於有視覺的活著目標，突然的強光使其失明片刻。
- 在挑戰中，此效果持續至目標下一回合結束。
- 若施放於受疾患苦楚的生物，此法術可將其疾患的進展降低一個階段。若這讓疾患的第一個階段得以解除，疾患即被消除（[危險要素](/crawl-phase/hazards/)）。
- 若施放於不死生物或荒域的靈體，此法術造成穿刺傷害。
- 若施放於不死生物，他們承受一點傷勢。
- 若施放於物品，物品發出明亮的光芒。
- 若施放於植物，植物迅速放大生長。
- 每額外花費一點決心，可多增加一個目標。
- 由於天火不產生熱量，術士可以點燃其法杖尖端或帽子以提供光明。
- 被天火點燃的物品在磨刀霍霍表上抽到「火把熄滅」結果時熄滅。
- 天火在大多數方面的表現如同正常火焰。可以用水或隔絕氧氣的方式撲滅。

### 生命

**材料：** 一根鳳凰的羽毛

此法術對活著且健康的生物無效。但若施放於瀕死狀態的對象，他們清除此狀態。該對象隨即承受壓力狀態。

- 每額外花費一點決心，可多增加一個目標。

### 封印誓約

**材料：** 一袋哥布林的牙齒和指節

術士使用此法術以魔法封印兩個自願方之間訂立的合約或誓言。

- 若誓約任一方違反誓約條款，兩件事情將會發生：
  - 誓約的所有其他方立即知曉此違約行為；且
  - 違約者命中注定：在未來最關鍵的時刻，他們在命運考驗中將自動承受大失敗。
- 命中注定不一定是死刑，但與不滿的其他方聯合起來，往往便是了。
- 每+1決心可將另一方加入誓約。

此法術永久生效，除非被解除。解除此法術會觸發第一個後果，但不觸發第二個。然而，一旦誓約被違反，這些魔法後果便無法以任何方式解除或移除。

### 真言

**材料：** 裝在金色聖體杯中的珍貴葡萄釀製的聖禮用酒

對某人施放此法術後，術士可以知道目標是否在知情的情況下說謊。

- 在法術效果期間，目標明白自己基本上是被連接到一台魔法測謊儀上。

此法術需保持專注。
