## 聖杯道途

地下世界開啟之際，前往追尋彌思里斯聖杯者多不勝數，一套龐大的支援體系因此應運而生，兼及世俗與宗教各方面。收留朝聖者的旅館與公眾客棧、供外地人兌換貨幣的錢幣商、販售補給與物資的商鋪、鑄造武裝的鐵匠舖、供虔誠信徒祈禱的禮拜堂與神祠、兜售偽造地圖與謠言的江湖騙子、為起草特許狀與遺囑而服務的書記與律師，以及為那些無功而返者療傷的藥劑師——這一切幾乎在一夜之間全部湧現。聖杯的符號因而成為這些行業商家的象徵，同時也具有神聖含義，至今仍被用來標示各類必要的服務。

**常見主題：** 驚悚派藥劑師、勇於冒險的地圖繪製師、爽朗的神職人員、膽大的考古學家、老練的理髮外科師、年輕的馴蟲師

## 聖杯天賦

### 煉金術

此天賦賦予你一個新的紮營行動：煉製煉金藥。當你煉製煉金藥時，你使用煉金套件，將你持有的任意數量試劑轉化為煉金物質。試劑從地下世界怪物的屍體上採集。煉金物質儲存在密封瓶中，分為三個類別：藥水、油劑和炸彈。詳見附錄 B 中關於煉金術的更多資訊。

### 馴獸師

此天賦允許你訓練並向動物夥伴下達命令。作為紮營行動，你可以教導一個動物夥伴一個命令。一般訓練的動物最多可學習三個命令。你可以用新命令重新訓練動物，替換其之前掌握的命令之一。

從以下命令清單中選擇（或 GM 批准的新命令）：

- 撲上去

- 去求救

- 靠攏

- 叼來

- 追蹤

- 狩獵

- 表演特技

- 守衛

- 待命

若你已精通此天賦，你擁有的一頭動物被視為你的使魔。使魔最多可學習五個不同的命令。此外，若你的使魔將要承受一個足以致命的傷勢，你可以花費決心，宣告牠只是處於垂死邊緣，並非真正死亡。

關於動物夥伴的更多資訊，[冒險者進階](/adventurer/advancement/)。

### 書蟲

你對所讀過的一切擁有近乎百科全書式的知識。一旦你使用過「閱讀書籍」紮營行動（[紮營階段](/camp-phase/)），你此後可以投注學識，就該書的主題提問。保留一份你已讀過的書籍清單。這些書籍在投注學識時視同主題，但不能在命運考驗中提供協助。

### 外科醫

你可以為夥伴提供治療。你可以消耗一份藥布來為某人治癒傷勢（[命運考驗規則](/basics/tests-and-fate/)）。此效果不能用於清除壓力狀態。

此外，此天賦賦予你一個新的紮營行動：外科手術。當你對一名未承受壓力狀態的公會夥伴施行外科手術，且該夥伴正在使用「休息恢復」紮營行動時，你可以治癒其所有傷勢。

### 忠告

在挑戰階段的任何時刻，你可以向另一名冒險者大聲建議，並從手中遞給那位玩家一張牌。該牌的花色必須與你建議他們採取的行動相符。另一位玩家可以打出那張牌，只要他們用來接受你的建議。在其他所有方面，它被視為一張普通的挑戰牌（[挑戰階段](/challenge-phase/)）。

若你在使用此天賦時花費決心，另一位玩家可以將此牌作為打斷打出。否則，每回合一張牌的標準規則仍然適用（[挑戰階段](/challenge-phase/)）。

此天賦每輪只能使用一次。

### 高詠

此天賦賦予你一個新的紮營行動：吟唱高詠。當你吟唱高詠時，為眾人朗誦至少兩節史詩詩篇。若你這樣做，你可以從小阿爾卡那棄牌堆中選取等同於你聖杯值數量的牌。這些牌稱為靈感牌。將牌分發給其他玩家——你可以為自己保留一張。每位玩家最多只能持有一張靈感牌。靈感牌可在挑戰階段作為行動打出。或者，當你進行命運考驗或推動命運時，你可以打出靈感牌替代從牌堆抽牌。靈感牌在使用後或本場次結束時失效，以先發生者為準。

### 學識大師

你是古代語言與學識的研究者。此天賦賦予三項優勢：

- 你通曉許多語言和方言。你被認為能對任何可能遭遇的口語語言具備初步流利的能力。

- 你能翻譯古代文字與盧恩符文。在探索過程中遭遇的短篇古代符文段落，翻譯需要一至兩個時段。翻譯較長文本（例如書籍）則需要一個城市行動。

- 當你投注學識時，你可以花費決心替代消耗一次學識投注。
