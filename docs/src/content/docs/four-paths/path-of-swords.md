寶劍天賦

神盾
若你持有一面完整的盾牌，且即將受到來自物理來源（攻擊、陷阱等）的效果，你可以選擇讓你的盾牌被缺口，而非承受該效果。

厄眼
在挑戰階段使用遠程武器時，你可以在自己的回合花費一張寶劍牌執行一次精準射擊。此攻擊自動命中，無視目標的先攻值。

重金屬機器
在挑戰階段，若你穿著鐵製或鋼製護甲，你可以使用一個雜項行動（棄置任意一張牌），將你的寶劍值加入先攻值，作為對抗單一行動的插斷。每輪只能使用此天賦一次。

怪物獵人
當你精通或磨練此天賦時，同時選擇一個仇敵類別和該敵人的專項。

元素獵人
你專注於對付自然的超自然化身。

專項：任何特定生物（例如：水精靈、火蜥蜴精、海妖、樹精、樹巨人）

人類獵人
你專精於獵殺最危險的獵物。「人類」一詞在此略有誤導，此類別涵蓋所有族裔、血脈和性別。

專項：任何特定族裔：精靈族、人類、地底民、獸人

靈魂獵人
你專注於對付來自遠方界域的異能存在。你能應對無形靈魂、顯形靈魂和附身靈魂。

專項：來自某一遠方界域的靈魂（即：荒域、林野界、詭異界、穹蒼界）

不死獵人
你擁有安撫不安死者的經驗。

• 你獲得一個名為「【仇敵】獵人」的主題。你可以用此主題投注學識了解這些生物，或如常在與其相關的命運考驗中獲得優勢。

• 你對你專項的仇敵進行攻擊時帶有優勢。

專項：特定不死類型（例如：骷髏、殭屍、幽靈）

巫術獵人
若某件事的答案是「某個術士幹的」，你就能應付它。

選擇一個仇敵和一個專項（如有新類型的生物讓你大動肝火，你可以透過城市行動更換你的仇敵）。

野獸獵人
你專精於獵殺、追蹤和誘捕各種形態大小的野獸。何謂「野獸」？凡是中世紀動物誌中收錄的，大概都算野獸。野獸原生於肉身界——它們饗宴、爭鬥、逃竄、繁衍。

專項：任何特定物種（例如：蝙蝠、蜈蚣、奇美拉、雞蛇怪、恐龍、老虎）

專項：任何特定類型的魔法顯形（例如：獸人型變種、魔像、黏液體）

GM注意
怪物獵人的各類別可能有所重疊。例如，一隻巨型甲蟲對野獸獵人和巫術獵人可能都是合適的專項。這沒有問題，無需在分類細節上鑽牛角尖。

掠奪者
在挑戰階段進行攻擊時，你可以同時朝目標移動一個區域（[衝突階段](/challenge-phase/)）。

雙手精專
若你在挑戰階段雙手持用一件近戰武器，你可以使用次要行動以錢幣牌或寶劍牌進行攻擊。

戰場故事
作為紮營行動，向你的公會講述你過去冒險的故事。也許這是你們曾在遊戲中進行過的冒險，或是你的背景故事中的往事。你——以及所有聆聽並稍加配合角色扮演的人——可以選擇以下其中一項收益：

• 為一條未充能的羈絆充能。

• 獲得額外一點決心；此可讓決心上限提升至5。

提醒
在角色創建期間，你將依 1 至 4 的順序排列你的屬性——寶劍、錢幣、聖杯或權杖。評等為 4（最高等級）的屬性，即成為你的道途。

你的道途決定了你最初可使用的天賦。你將以一個精通的道途天賦和其餘磨練中的道途天賦開始遊戲。關於學習、磨練和精通天賦的詳細資訊，[冒險者進階](/adventurer/advancement/)。
