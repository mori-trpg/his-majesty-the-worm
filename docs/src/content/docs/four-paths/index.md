第五章

四條道途

「與怪物搏鬥者，謹防自身亦成怪物。
若你凝視深淵過久，深淵亦將回望你。」

──弗里德里希·尼采，《善惡的彼岸》

此處有一則謎題。一個愚者站在四位國王之間。愚者手持一把修長而鋸齒狀的寶劍。四位國王手無寸鐵。

錢幣之王隨即開口。他是個精明的人，富可敵國。

權杖之王率先發言。他是個邪惡而強大的術士。

「如果你殺死其他幾位，我將施放一道咒術，讓世上最美麗的女子對你芳心暗許。她將成為你的新娘，憑我的法術，你們將兒孫滿堂。」

聖杯之王繼而開口。他是個虔誠的人。

「呸！哼！哼！我有足夠的黃金買下一個女人和教團的恩寵。別廢話了。把那幾個乞丐捅死，你將獲得一輩子都花不完的黃金。」

最後開口的是寶劍之王——四人之中最遲鈍卻也最強大的。

「這有什麼好爭的？你手裡有把劍，用它！若你以我之名揮劍，我將封你為騎士。上述所有賞賜，皆是你應得的。」

「吾乃彌思里斯秘義的進階傳人，以一元神的真聲示人。殺死其他幾位，吾將傳授你深邃的智慧。」

愚者將會怎麼做？

寶劍道途

通往地下世界的最初探索是一段血腥的往事。正是一群石膏礦工發現了通往第一層的入口。他們好奇地鑿開了前廳。如同盜墓者，他們發現了金碧輝煌的寶藏和堆積如山的古幣。然後，他們遇見了格魯。無人生還。

在第一次十字軍東征終於動員之前，傷亡人數已悽慘地節節攀升，因為地下世界的誘惑對勇敢者與魯莽者而言都難以抗拒。隨著地下世界逐漸揭開面紗，越來越多的生物從半遺忘的傳說中蛻變為現實。奇異的野獸與惡意的存在爬出洞開的密室，潛入恐懼籠罩的城市。教團起初深陷於地下世界帶來的現實考量與神學難題之中，後以激昂的演說號召人類驕傲而古老的各大家族團結一心。在教團的領導下，第一次十字軍東征清剿了逃入城市的怪物，並在地下世界的門檻前紮營駐守，守護人類的土地免遭進一步侵害。

直到第二次十字軍東征期間，人們才發現，死在地下世界的人並不會真正死去……
