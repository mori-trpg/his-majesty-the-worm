權杖道途

城市與魔法的關係，長久以來充滿了血腥與哀愁。在第一帝國時代，法術被用作崇拜鴉首女王的儀式行為——她是最早被神格化的女帝之一。在共和時代，法術被視為與保皇派和精靈族根本相連，因而被列為違法。儘管如此，仍有數個秘密結社在暗中蓬勃發展。共和時代因捕巫者愈演愈烈的狂熱而著稱，他們竭力搜查並剷除這些潛藏的術士。

在地城時代降臨後，全面禁絕法術被證明是不可能的。地下世界中有太多考驗與敵人在本質上是神秘的。法術的修煉成為了一種必要之惡，教團開始起草相應的特赦令與贖罪券。

時至今日，在城市中修煉法術者必須取得執照並向王室登記。官方術士由靈知塔督守授予職權杖，並獲准加入公會探索地下世界。私自修煉法術可被處以罰款，但只要不在城市境內施法，通常不受管制。儘管如此，捕巫總監仍是城市政治中頗具影響力的人物，其麾下的聖堂武士對不受管制的法術抱持著更為嚴峻的態度。

權杖天賦

反制法術
你可以憑藉意志抵消法術的效果。此天賦有兩種應用方式：

• 你可以阻止敵方術士在戰鬥中施法。在挑戰階段，若你能察覺到一名術士正在施法，你可以花費決心，以插斷方式吟誦咒文。

• 若你的吟誦咒文值高於敵方術士的總值，其法術將消散失效。

• 若你的吟誦咒文值等於或低於敵方的總值，法術同樣消散失效，但你必須查詢禍兆表（附錄 A，[法術規則](/sorcery/rules/)），因為魔法出現了偏差。

• 你可以中止已在發動中的法術。你可以花費決心，立即終止一個正在持續效果中的法術。

幽法
沐浴在秘法藝術之中，你可以施展小規模（但有意義的）魔法。你可以施展以下細微效果：

• 你可以讓你所在區域內一個重約十磅、能舒適置於單手的物體漂浮。你對此效果沒有精細的運動控制，但可以將物體帶向你、抬起，或推開。

• 你可以召喚出一個招搖、無害且顯然是魔法的幻象——一陣閃爍的火花、伸展在雙手之間的彩虹、你初戀的虛影。

• 你召喚出一個能單手持握之物的簡易幻象。任何試圖與此物互動的人都會意識到這是幻象，但其外觀相當逼真。

此外，花費決心，你可以集中靈視一個時段。靈視使你能看見隱形和遮蔽的事物。你能看穿任何幻象的真實形態。你能自動辨別事物是否具有魔力或受到了魔法影響。你也能辨別他人是否為術士。

在挑戰階段使用此天賦視為雜項行動。

法術
走在權杖道途的人是術士。施法與法術修煉的詳細說明，見附錄 A：法術。

秘法術
你已學會如何汲取遠方界域的能量。使用此天賦，你可以透過一根靈木權杖，向最遠一個區域外釋放一道魔法能量爆發，以武器方式造成傷害。在挑戰階段，以你的權杖進行攻擊計為權杖行動（[衝突行動](/challenge-phase/actions/)）。

此外，此天賦賦予你一個新的紮營行動：訂立誓約。當你訂立誓約時，從下一頁中選擇遵守一個或多個誓約。

荒域魔法
你可以施展荒域的法術。荒域是一個充滿死亡與腐敗的陰暗之地。此界域的法術涉及不死之事、散播恐懼，或造成傷害。

林野界魔法
你可以施展林野界的法術。林野界是一個蒼翠而蠻荒之地。此界域的法術影響元素、指使野獸，並改塑自然世界。

詭異界魔法
你可以施展詭異界的法術。詭異界是一個夢境之地，不受理性觸及。此界域的法術影響思維與情感、召喚幻象，並扭曲現實。

穹蒼界魔法
你可以施展穹蒼界的法術。穹蒼界是一個光明而神聖之地。此界域的法術涉及天使、提供治療與保護，並召喚令人敬畏的神蹟。

誓約

當你使用「訂立誓約」紮營行動時，每訂立一個誓約，為你選擇的一個法術材料充能。此充能可代替花費決心來施放法術，但若要賦予法術更強的威力，仍須額外花費決心。

每個材料只能充能一次。你的誓約持續至其被打破，或直到法術被施放為止。材料保持充能狀態，直到法術被施放，或直到你打破誓約。若你打破誓約，查詢禍兆表（附錄 A，[法術規則](/sorcery/rules/)）。

吞噬活物
你只能食用在入口之際仍然活著的生物：蛆蟲、蝦子、老鼠、幼嬰。若你沒有攜帶這些生物（它們難以攜帶與保存），你可以在紮營行動中以聖杯進行命運考驗，看看能否在地下世界中找到。若你失敗，你被視為飢餓狀態（[紮營階段](/camp-phase/)）。若你成功，你找到了足夠的小蟲，能讓自己吃得飽足。

棄盔甲
無論其材質如何，你不得穿戴任何護甲或頭盔，也不得使用任何類型的盾牌。

棄獸皮
你不得穿戴任何野獸的皮或毛皮，或其任何衍生物（例如皮革）。你可以穿著亞麻或棉製的編織服飾。

棄武器
你不得接觸、取用、操作或使用任何種類的武器——哪怕是當作棍棒使用的擀麵棍也不行。你所擁有的武器必須以布包裹並綁在背包上。有趣的是，權杖不計為武器。

棄木材
你不得使用、觸碰或攜帶任何由砍伐木材製成之物。你不得攜帶火把、盾牌或棍棒。若所用木材是在城市中以黃金購買的（因為那木材必然是砍伐所得），你便不能享受篝火的取暖之利——你必須坐在火光與熱氣之外。由於權杖是以出於自願的靈木製成，不適用此禁令。

暴食
你必須吃超過你應得的份量。每當你消耗口糧時，你必須吃雙倍的正常份量（兩份口糧而非一份），否則即違反此誓約。

遮面
你必須隨時以面具遮住面容。術士可能選擇瓷器、布料、木材、骨骼或更奇異材質製成的面具。佩戴面具會在所有社交考驗中帶來劣勢，GM 也會據此調整你的社交遭遇。

苦行
你必須購買並穿著鬃毛苦衣，套在正常衣物之下。這使你承受壓力狀態。若你脫下苦衣並清除壓力狀態，即違反此誓約。

刻符
訂立此誓約時，你在自己身上刻下盧恩符文。這使你承受一點傷勢。若你治癒此傷勢，即違反此誓約。

緘默
若必須開口，你只能以耳語說話。人們必須與你非常靠近，且環境必須非常安靜，才能聽清你的話語。若你大聲說話，即違反此誓約。

誠信
你不得說出知情的謊言。若你這樣做，哪怕只是玩笑，此誓約即告破滅。

左右為難……
有時，誓約可能使你陷入必須打破一個誓約才能履行另一個誓約的困境。這根植於神話傳說之中。想想庫夫林——他不能拒絕陌生人提供的食物，也不能吃狗肉。當一個陌生人提供他狗肉時，他無論如何都無法脫身。

我們建議不要迴避這些衝突。若遊戲中的情節要求兩個誓約相互矛盾，那就讓它發生。這最終不會有多痛苦。讓它自然發展，看看會發生什麼。
