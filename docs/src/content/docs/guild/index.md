「我們為何前往偏遠之地？是為了證明我們的冒險精神，還是為了講述不可思議的故事？我們去那裡，是為了在朋友之間獨處，在沒有人跡的土地上找到自我。」

──喬治·馬洛里，引自《最荒野的夢：喬治·馬洛里傳》

第一次十字軍東征之後，城市面臨了一個新的難題：如何處理大量湧入的財富。覆蓋林木的尖塔頂端已是真正的鍍金。可怕的怪物在被遺忘王朝遺留的古老金幣上沉睡。返程十字軍帶回的龐大財富既是巨大的恩賜，也是棘手的問題。

A

乞丐王子——其實離乞丐遠得很——從十字軍東征歸來時，無意中重創了銅幣與銀幣的幣值。臍帶市集的物價大幅膨脹。為了整頓市場，他向放貸人以高利借入所有能存放的黃金，創立了中心銀行。

正是暴君皇帝對所有從地下世界帶回的金幣課征 50% 的稅。每掠奪兩枚金幣，就有一枚入了皇家金庫。

非流動資產，如寶石、精緻衣物或藝術品，則可不繳稅地帶到地面：是豐盛的美麗，用以充實城市。

為了更好地管控金幣從地下世界流出，暴君皇帝進一步試圖掌控誰能下去、誰能上來。他頒布法令，要求所有冒險者向教團登記姓名，以及某種可用於辨識身份的標誌或符號。這促成了公會的形成——冒險者以集體單位組建，並借鑒了人類的紋章習俗。

當獅鷲王掌權之後，許多（極其富裕的）冒險者請求他廢除暴君的法令——或至少降低稅率。

他沒有。

公會名冊

你與其他冒險者在冒險召喚期間共同組成一個公會。正如你用冒險者記錄單來追蹤你的冒險者，全桌玩家透過公會名冊來追蹤公會（印於後續幾頁）。公會名冊可從 hismajestytheworm.games 下載。

3

徽記

根據公會名稱，玩家共同選擇公會的印記、紋章或肖像。

精靈族公會偏好抽象印記：由形狀組成的萬花筒，大致對應某種自然現象，如山脈、有意義的花朵或吉祥星象。地底民公會徽記以精美書法層層疊加意涵。獸人與人類則偏好動物與怪物的紋章圖案，以喚起危險、幽默或力量的感受。

桌上最有美術才能的玩家，應在公會名冊記錄單的指定欄位繪製一個徽記或公會素描。

1

公會名稱

所有玩家共同決定公會名稱。這將在城市中把你的公會登記為合法實體，使其有權前往地下世界。

選擇一個能喚起你們公會主題、代表公會共同目標的名稱。最重要的是，選一個很酷、讓每個人都覺得好玩的公會名稱。

公會名稱範例：四張王牌、月之侍女、鐵甲蛇隊、侍女團、正義之子、戰鬥信仰、破桶者、奇美拉公司、焚旗兄弟會、陛下的深淵探掘者、劍姐妹、尋覓者、忙碌蜜蜂

2

契約條款

公會名冊定義了你們公會的戰利品分配規則。預設契約規定，所有公會成員平等分配戰利品，葬禮費用由公會支付。

填寫公會名冊時，花點時間討論你們公會的戰利品分配規則：誰分得最多份額、費用如何分攤、倒下的冒險者裝備如何處置等等。

現行契約

地下世界既有怪物，也有寶藏。強大、有影響力且富有的人渴望得到地下世界中的東西。他們提供契約：以風險換取報酬。替他們完成地下世界中的任務，他們也會讓你富起來。

當你的公會接受契約時，在公會名冊記錄單的「現行契約」欄位中記錄下來。

當你履行契約時，公會中每個人獲得 1 XP。

在城市中創造有趣角色並提供有趣契約是 GM 的職責；詳見第 160 頁。

公會在同一時間只能追求一個任務，但可以同時擁有任意數量的契約，以合理為限。然而，未能見到成果的委託人可能會撤回契約、詆毀公會，或親自出手解決。

一般來說，完成契約是一項比完成任務更小且更獨立的工作。一個任務可能需要多次下探地下世界；通常，契約只需一到兩次。

事蹟與聲望

當地下世界發生了重大事蹟，整個城市都會沸騰。冒險者是大生意——完成壯舉的公會將開始吸引重要人物的注意。你的公會完成的每一項獨立成就都被記錄為事蹟。你在城市階段步驟 2（第 149 頁）中，在公會名冊上追蹤事蹟。

你的公會聲望等於你已完成的事蹟數量。聲望是衡量你的公會獲得多少流言與知名度的抽象指標。聲望等級為 0–5。GM 會在你目前的聲望分數脈絡下框架角色扮演的互動。
